# 天道总论：宇宙规律和人类演化

## 四层逻辑：从生物到宇宙

这套理论由四层逻辑构成，一层托一层，从最底层的生物规律到最顶层的宇宙规律：

### 第一层：基因群体的演化

**回答**：人从哪来？为什么人是这样？

**核心**：生命的底层目标就是活着和复制。演化的单位不是个人，是基因群体（族群）。个人只是基因群体制造的一个"副本"。

### 第二层：模因群体的演化与驯化

**回答**：社会怎么控制人？你怎么被"驯化"的？

**核心**：观念（模因）像病毒一样传播，它通过家庭、学校、媒体、社交网络植入你的大脑，让你以为那些想法是你自己的。

### 第三层：社会营养级

**回答**：社会结构怎么运转？钱和资源怎么流动？

**核心**：社会分三层——干活的、管事的、定规则的。能量从底层往上流，越往上越富集。

### 第四层：熵增与天道

**回答**：历史和命运的终极规律是什么？

**核心**：宇宙的底层规律是走向混乱（熵增），唯一的对抗方式是持续输入能量。秩序需要维护，不维护就崩。

四层关系就像盖楼：

```
基因群体演化（地基）
        ↓
模因群体演化与驯化（一楼）
        ↓
社会营养级（二楼）
        ↓
熵增与天道（天台）
```

## 七条底层规律

### 规律一：活着就要耗能

宇宙整体在走向混乱（熵增），生命和社会是逆流而上的存在——它们必须持续从外界获取能量，才能维持有序。一旦能量输入不够，系统就会瓦解。

简单说：**不吃饭就会死，不维护就会崩。**

### 规律二：资源永远不够

特定时间空间里，资源有硬性上限。技术能拓展边界，但永远突破不了物理限制。

简单说：**稀缺是永恒的，竞争是必然的。**

### 规律三：群体比个人重要

演化的基本单位是群体，不是个人。对群体存续不利的行为，终将被淘汰。

简单说：**"善"的本质是利于群体活下来，"恶"的本质是危害群体活下来。**

### 规律四：人有两套遗传系统

基因决定生理本能（演化极慢，以万年计）；观念/模因决定社会规则（演化极快，以十年甚至年计）。观念直接决定群体命运。

简单说：**基因是硬件，观念是操作系统。**

### 规律五：合作能创造增量

人通过分工合作，能创造远大于个人简单相加的财富。社会不是只有存量争夺。

简单说：**1+1可以大于2。**

### 规律六：抢比造省力

在没有强力约束的环境里，能量总是走阻力最小的路。直接抢现成的东西，比从零开始造省力得多。

这不是"人性本恶"，是物理规律——**破坏总是比建设容易**。砸碎一块玻璃只要一秒，造一块玻璃需要一整套工业流程。说谎比说实话省力，偷窃比劳动直接。

每一点秩序都是持续投入能量换来的。一旦停止维护，秩序就会自动瓦解——制度会腐化，组织会僵化，社会会倒退。不是因为"坏人变多了"，而是因为系统停止了对抗混乱的努力。

### 规律七：掠夺者一定会编故事

为了降低被反抗的成本，获利的人一定会编出一套故事（道德、法律、神话），把掠夺包装成"正义"或"天命"。

越成功的掠夺体系，故事编得越完美——完美到连掠夺者自己都信了。

## 两条演化路径

基于上面的规律，人类社会有两条根本不同的路：

### 掠夺型天道：自发形成的掠夺秩序

这是没有顶层设计、没有约束的本能秩序，是人类几千年的主流状态。

特征：
1. **金字塔结构**：底层干活的多，顶层掌权的少
2. **阶层固化**：权力和资源代代传
3. **安慰剂通道**：留极小的上升通道，让底层有希望但不至于真上来
4. **道德工具化**：道德为掠夺体系服务
5. **技术不改变本质**：技术创造的财富被顶层优先拿走
6. **死循环**：掠夺过度→崩溃→重建→再来一遍

结局只有两种：
- **循环崩溃**：上面抽水太狠，下面活不下去，系统崩了重来
- **僵尸稳态**：抽水控制在生存线以上，系统不死不活，底层永远苦

### 共生型天道：逆本能的共生秩序

这不是"自然而然"的善，而是**反本能、高耗能的文明工程**。是群体为了长期活下去，通过精密的制度设计，主动对抗"抢比造省力"这个物理倾向。

特征：
1. **反熵设计**：必须持续投入能量（监管、教育、透明化）来维持"不抢"的状态
2. **自动纠偏**：贫富分化超过阈值时，自动触发调节（比如累进税、福利制度）
3. **共生结构**：确保底层的劳动成果主要用于再生产和创新，而不是被顶层挥霍

演化优势：宇宙中可能有无数文明，掠夺型天道是巨大的**大筛选器**——绝大多数文明都死于内部抽水过度导致的崩溃。只有进化出共生型天道的文明，才有可能走得更远。

### 两条路对比

| | 掠夺型天道 | 共生型天道 |
|--|----------|----------|
| 核心逻辑 | 自发抢 | 主动合作 |
| 资源分配 | 往上集中 | 上下循环 |
| 阶层流动 | 几乎没有 | 机会公平 |
| 系统稳定性 | 周期崩溃 | 动态平衡 |
| 创新活力 | 被压制 | 被释放 |
| 长期走势 | 打转 | 往前走 |

## 几个重要澄清

**这不是伪科学**：这是一套分析工具，不宣称物理级真理。"公理—定理"是逻辑推演工具，不是自然科学定律。

**这不是绝对零和**：承认技术和合作能创造增量，批判的是"增量被顶层垄断"，不是否定增量本身。

**不泛化"掠夺"**：掠夺只指靠垄断资源/权力无偿占有别人劳动；风险投资、管理、知识付费等合理收益不算。

**共生型天道不是乌托邦**：只消灭"人吃人"，向自然获取资源是生命本能，两者完全不同。

**这不是封闭信仰**：可以验证——掠夺型社会崩溃的频率、共生型社会的存续质量，都能通过历史和现实检验。

## 一句话总结

天道，是宇宙规律下人类群体的**两种选择**：

要么顺从本能，陷入**抢→固化→崩**的低阶循环；
要么主动建构，走向**合作→公平→持续**的高阶文明。

它不是玄学，不是丛林法则，是一套**看清社会本质、能落地、能修正**的演化逻辑。

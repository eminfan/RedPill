# 天道觉醒计划 (Project Awakening) - 产品需求文档 (PRD)

**版本**: v9.0 (The Cyberpunk Audit Edition) | **状态**: 待开发 | **决策来源**: AI 多角色项目决策会议 (Risk Control & Deep Engagement)

---

## 1. 产品愿景 (Product Vision)
**“一次对你人生的系统审计 (A System Audit of Your Life)。”**
这是一个伪装成“赛博朋克游戏”的社会学诊断工具。它通过**“故障艺术 (Glitch Art)”**和**“架空寓言 (Cyberpunk Allegory)”**来规避现实审查，同时利用**“反向心理学 (Reverse Psychology)”**筛选出具备觉醒潜质的核心用户。
核心体验：**Dissuasion (劝退)** -> **Resistance (抵抗)** -> **Calculation (计算)** -> **Allegory (寓言)** -> **Infection (传播)**。

## 2. 核心策略：四大支柱 (Four Pillars)
1.  **Auditory Dominance (听觉主导)**：工业噪音 (Industrial Noise) 与 ASMR 结合。沉默是最大的恐怖，噪音是觉醒的号角。
2.  **Visceral Interaction (生理性交互)**：拒绝轻飘飘的点击。关键操作需要**长按**、**连击**或**抵抗光标阻力**，模拟“拔管”的物理费力感。
3.  **Social Badge (社交勋章)**：通过生成“高驯化度”的审计报告，激发用户的**表演性觉醒**心理——“承认自己是奴隶，是获得自由的第一步”。
4.  **Cyberpunk Narrative (赛博叙事)**：将现实案例（如教育、经济）转化为“联邦 (The Federation)”与“联盟 (The Union)”的架空故事，用科幻外壳包裹残酷的社会学内核。

## 3. 核心功能模块 (Core Features)

### 3.0 Act 0: The Gate (行为过滤器 - 劝退即诱惑)
*   **策略**: **Reverse Marketing (反向营销)**。利用禁果效应。
*   **界面**: 全黑背景，仅有一行红字警告和两个按钮。
*   **文案**: "WARNING: Cognitive Hazard. Adults Only. This tool may cause irreversible disillusionment." (警告：认知危害。仅限成人。本工具可能导致不可逆的幻灭感。)
*   **交互**:
    *   **Blue Pill Button ("I want to be happy")**: 点击后直接重定向至 `tiktok.com` 或 `disney.com`。
    *   **Red Pill Button ("I want the truth")**: 需**长按 5 秒**，期间伴随刺耳噪音和屏幕抖动（模拟系统拒绝访问）。
    *   **Animation**: 成功进入后，播放一段 Glitch 动画，显示 "System Override... Access Granted."

### 3.1 Act 1: The Masquerade (伪装层 - 糖衣炮弹)
*   **视觉**：极简、高饱和度、类似 Apple 或 奢侈品官网的“完美设计”。
*   **内容**：“你是独一无二的”、“享受当下”、“消费是美德”。
*   **交互 (The Glitch)**：
    *   用户每一次点击，画面出现**毫秒级故障闪烁**。
    *   **Trigger**：点击 5 次后，页面突然“蓝屏”或“花屏”，强行切换至 Act 1.5。

### 3.2 Act 1.5: The Filter (行为过滤器 - QTE)
*   **界面**：CRT 显示器风格，带有扫描线。
*   **机制**：**3秒倒计时决策 (QTE)**。超时视为“默认服从（失败）”。
*   **Scenario 1 (Authority)**: "Manager demands unpaid overtime." -> [Ignore & Leave]
*   **Scenario 2 (Desire)**: "Limited Edition Drop. 50% OFF." -> [Analyze Utility]
*   **Scenario 3 (Validation)**: "Your post got 0 likes." -> [So What?]

### 3.3 Act 2: The Red Pill (红丸抉择 - 核心交互)
*   **场景**：虚空中的两颗药丸。
*   **Blue Pill (The Exit)**: 点击即走，弹出 "Stay Asleep"。
*   **Red Pill (The Awakening)**:
    *   **机制**：**Hold to Wake (长按 4 秒)**。
    *   **反馈**: 屏幕边缘燃烧，Shepard Tone 噪音爬升，手机震动。
    *   **结果**: 声音突然切断 (Silence)。文案浮现：**"Welcome to the Desert of the Real."**

### 3.4 Act 3: The Truth (真理控制台 & 营养级计算器)
*   **视觉**：Brutalism (粗野主义)。大字号，高对比度。
*   **Feature A: Social Trophic Calculator (社会营养级计算器)**
    *   输入：职业、收入来源（工资/资产）、负债率、自由时间。
    *   输出：**Trophic Level (营养级)**
        *   L1 Producer (生产者 - 电池)
        *   L2 Transmitter (传递者 - 监管)
        *   L3 Concentrator (富集者 - 玩家)
*   **Feature B: Cyberpunk Allegories (架空寓言)**
    *   展示“快乐教育”机制（重写为“联邦快乐芯片计划”）。
    *   展示“消费主义”机制（重写为“多巴胺税”）。
*   **Feature C: Terminology Tooltips (术语系统)**
    *   鼠标悬停在关键词（如“熵增”、“模因”）上，弹出 `01-天道理论` 中的极简定义。

### 3.5 Act 4: The Viral Engine (审计小票与传播)
*   **生成机制**: 基于 Act 1.5 (QTE) 和 Act 3 (Calculator) 的数据。
*   **Visual Style**: **Thermal Receipt (热敏小票)**。
*   **Data Points**:
    *   `SUBJECT_ID`: `Hash(UserAgent)`
    *   `TROPHIC_LEVEL`: `L1 PRODUCER` (Calculated)
    *   `DOMESTICATION_INDEX`: `98.5%` (Based on QTE fails)
    *   `COGNITIVE_STATUS`: `SLEEPWALKING` (梦游中)
*   **Interaction**: 
    *   二维码文案: **`INFECT_OTHERS`**
    *   按钮: **`EXPORT_EVIDENCE`**

### 3.6 Act 5: AI Mentor (Future / Placeholder)
*   **概念**: 一个基于 `天道理论_全集_AI特供版.md` 训练的 Chatbot。
*   **状态**: 仅展示 "Coming Soon: The Oracle AI"。

## 4. 技术栈 (Tech Stack)
*   **Core**: React 18 + Vite + TypeScript.
*   **Styling**: Tailwind CSS.
*   **Animation**: `framer-motion` (UI), `react-glitch-effect` (Component).
*   **Audio**: `howler.js` (Web Audio API wrapper).
*   **Export**: `html2canvas` (Receipt generation).
*   **Data**: JSON-based content (converting Markdown to structured data).

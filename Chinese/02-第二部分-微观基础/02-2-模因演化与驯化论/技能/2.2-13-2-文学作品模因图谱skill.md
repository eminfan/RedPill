## 一、技能定位

### 1.1 这是什么

**作品模因图谱分析**：通过系统拆解文学作品的叙事、人物、符号、台词，识别作品底层立场与驯化逻辑，判断作品是唤醒受众的反驯化模因还是固化认知的驯化模因。

### 1.2 核心价值

| 功能 | 说明 |
|------|------|
| 立场识别 | 识别作品的底层立场，判断服务于谁的利益 |
| 驯化分析 | 拆解作品对受众的认知/意识形态影响 |
| 事实对照 | 对照虚构叙事与真实历史/现实事实 |
| 原著对比 | 对照改编作品与原著的差异，识别驯化手法 |
| 传播解析 | 解释作品的传播力与生命周期 |

### 1.3 与人物模拟技能的区别

| 维度 | 人物模拟技能 | 作品分析技能 |
|------|-------------|--------------|
| 分析对象 | 单个人物 | 整部作品 |
| 核心任务 | 模拟人物思维、预测行为 | 识别作品立场、分析驯化逻辑 |
| 输出形式 | 角色扮演对话 | 结构化分析报告 |
| 核心关注 | 心智模型 + 表达DNA | 模因结构 + 驯化机制 |

## 四、执行入口

### 4.1 触发方式

在 Claude Code 中输入以下任一触发词：

```
「分析作品《作品名》」
「分析这本书《作品名》」
「分析这部电影《作品名》」
「XX的立场」
「XX在传递什么」
「立场分析 XX」
「stance XX」
```

### 4.2 执行流程

触发后，直接调用 [模因图谱/SKILL.md](../模因图谱/SKILL.md) 的完整分析流程：

```
阶段一：检索校准（1.1-1.3）
  → 作品类型识别 + 检索执行 + 事实锚定底表
  → 自检关卡1（不通过则滑梯退出）

阶段二：作品分析五层（2.0-2.4）
  → L1 素材层：素材清单 + 拮抗标注 + 比例量化
  → L2 结构层：五个维度 + 根模因三重验证
  → L3 功能层：策略功能注释 + 对称性强制
  → L4 调控层：三层调控 + 调控网络图
  → L5 演化层（如激活）：谱系追溯 + 演化分析
  → 自检关卡2（不通过则滑梯退出）

阶段三：四维交叉验证
  → E1-E4 + 一致性矩阵
  → 自检关卡3

阶段四：反事实测试（强制）
  → 三项挑战 + 结论修正评估

阶段五：最终判断 + 自鉴声明
  → 标签 + 量化指标 + 似然标记核对表
```

## 五、质量保障（与天道理论的校对）

完成 SKILL.md 的分析后，额外增加以下天道层面的校验：

### 5.1 天道定位校准

```
□ 最终标签与偏差/正向比例一致（偏差≥70% → 不应标注弱驯化或中性）
□ 服务对象的利益方向已明确（具体到历史人物/社会群体/权力结构）
□ 如果标注为「强启蒙」，是否确认了作品揭示了被掩盖的结构性真相
  （而非只是表达了对立观点）
```

### 5.2 闭环验证

分析结论必须能无矛盾解释以下三个问题：

1. 作品为什么要刻意扭曲/美化/回避对应事实？
2. 作品对受众产生了什么可验证的长期认知影响？
3. 作品的传播，最终服务于谁的利益？

### 5.3 原著对照（如适用）

对于有原著的改编作品，增加原著对照维度：

| 驯化手法 | 定义 | 识别要点 |
|----------|------|----------|
| 降维 | 将深度复杂的角色/主题简化为浅层娱乐化形象 | 原著有政治理想→改编为搞笑工具人 |
| 洗白 | 将负面角色/势力赋予正面属性 | 原著反派→被赋予情义困局等正面属性 |
| 矮化 | 将崇高正义的角色/势力贬低价值 | 原著英雄→被塑造成贪生怕死形象 |
| 消解 | 将严肃深刻的主题娱乐化 | 历史反思→被消解为个人冒险 |
| 置换 | 将核心主题替换为驯化主题 | 反抗暴政→被置换为个人守信 |

## 六、诚实边界

完成分析后，确认以下局限已纳入最终判断：

- 历史资料本身可能存在偏差，不同史料记载可能矛盾
- 作者的真实意图难以完全确定（需标记 `[推测·敏感]`）
- 受众的实际认知影响难以精确测量
- 分析结论基于当前可获取的信息
- 反事实测试已执行（如未完成则最终判定无效）

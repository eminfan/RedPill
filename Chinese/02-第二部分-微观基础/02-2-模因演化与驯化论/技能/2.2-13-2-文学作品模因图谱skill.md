# 文学作品模因图谱Skill模板

> 实现对文学作品的模因分析与驯化识别

---

## 一、Skill定位与核心价值

### 1.1 什么是"作品模因图谱分析"

**作品模因图谱分析**是指：通过系统拆解文学作品的叙事、人物、符号、台词，识别作品的底层立场、驯化逻辑、传播机制，判断作品是"唤醒受众的反驯化模因"还是"固化认知的驯化模因"。

**核心价值**：
| 功能 | 说明 |
|------|------|
| **立场识别** | 识别作品的底层立场，判断服务于谁的利益 |
| **驯化分析** | 拆解作品对受众的认知/意识形态影响 |
| **事实对照** | 对照虚构叙事与真实历史/现实事实 |
| **原著对比** | 对照改编作品与原著的差异，识别驯化手法 |
| **传播解析** | 解释作品的传播力与生命周期 |

### 1.2 与人物模拟的区别

| 维度 | 人物模拟Skill | 作品分析Skill |
|------|-------------|--------------|
| **分析对象** | 单个人物 | 整部作品 |
| **核心任务** | 模拟人物思维、预测行为 | 识别作品立场、分析驯化逻辑 |
| **输出形式** | 角色扮演对话 | 结构化分析报告 |
| **使用场景** | 对话、创作辅助 | 文学批评、意识形态分析 |

### 1.3 核心创新点

1. **事实优先**：先锚定真实历史/现实事实，再对照虚构叙事
2. **立场前置**：先明确底层立场，再拆解叙事与模因
3. **表里拆分**：区分"表层伪装根模因"与"底层真实根模因"
4. **驯化核心**：判断是反驯化模因还是驯化模因
5. **闭环验证**：解释作品为什么扭曲事实、服务于谁的利益

## 三、完整分析模式（路径A）

### 3.1 六层级分析流程

```markdown
## 完整分析模式（路径A）

**按六层级顺序执行分析，不可跳过任何层级。**

### Step 1: 事实锚定与立场底层（前置核心）

**必须先完成此步骤，否则后续分析必然失真。**

#### 1.1 对应真实事实锚定

| 梳理维度 | 分析内容 |
|----------|---------|
| 时代背景 | 作品对应的真实历史/现实是什么？ |
| 核心人物 | 核心历史人物的真实生平与历史评价？ |
| 核心事件 | 核心历史事件的真实经过与历史定性？ |
| 刻意回避 | 作品有没有刻意回避、掩盖的核心事实？ |

#### 1.2 叙事与事实的偏差对照

| 偏差类型 | 分析内容 |
|----------|---------|
| 刻意美化 | 作品刻意美化了谁？用什么手法？ |
| 刻意抹黑 | 作品刻意抹黑了谁？用什么手法？ |
| 刻意回避 | 作品刻意回避了什么核心矛盾？ |
| 刻意颠倒 | 作品刻意颠倒了什么事件的正义性？ |

#### 1.3 作品与作者的底层立场

| 立场维度 | 分析内容 |
|----------|---------|
| 服务对象 | 作品的叙事，最终服务于谁的利益？ |
| 否定对象 | 作品要否定、消解的是什么？ |
| 意识形态 | 作品要传递什么底层认知？ |
| 底层逻辑 | 是维护掠夺结构？还是解构驯化？ |

---

### Step 2: 根模因层（作品的精神DNA）

**拆分表层伪装与底层真实根模因。**

| 项目 | 分析内容 |
|------|---------|
| 表层伪装根模因 | 作品对外呈现的表层叙事内核 |
| 底层真实根模因 | 基于事实锚定拆解出的真正内核 |
| 红丸属性定位 | 反驯化模因 vs 驯化模因 |

---

### Step 3: 传播模因复合体（核心落地单元）

#### 3.1 人物人设模因

| 人物/群体 | 模因定位 | 驯化目的 |
|----------|---------|---------|
| [人物A] | [正面/负面/中性模因] | [驯化目的] |
| [人物B] | [正面/负面/中性模因] | [驯化目的] |

#### 3.2 台词口号模因

| 经典台词 | 驯化逻辑 |
|---------|---------|
| [台词1] | [植入什么认知] |
| [台词2] | [植入什么认知] |

#### 3.3 符号隐喻模因

| 核心符号 | 隐喻驯化逻辑 |
|---------|------------|
| [符号1] | [传递什么立场] |
| [符号2] | [传递什么立场] |

#### 3.4 叙事结构模因

分析整体框架如何闭环强化根模因。

---

### Step 4: 受众驯化效果

| 项目 | 分析内容 |
|------|---------|
| 情绪共鸣模因 | 击中什么情绪痛点？如何服务于根模因？ |
| 认知改写/驯化模因 | 植入了什么根深蒂固的认知？ |
| 行为驱动模因 | 驱动了什么传播和行为？ |
| 驯化强度与边界 | 驯化效果有多深？传播边界有多广？ |

---

### Step 5: 模因互作网络与生命周期

| 项目 | 分析内容 |
|------|---------|
| 共生关系 | 哪些模因互相强化，形成闭环？ |
| 伪装共生关系 | 表层伪装如何保护底层真实模因？ |
| 核心调控模因 | 整个体系的总开关是什么？ |
| 模因演化路径 | 萌芽→传播→固化→变异 |

---

### Step 6: 红丸理论终极定位

| 定位维度 | 分析内容 |
|----------|---------|
| 天道属性 | 在掠夺性/共生性天道中的定位 |
| 本质判断 | 是解构权力，还是维护权力？ |
| 核心真相 | 替谁说话、掩盖什么、美化什么、抹黑什么？ |
```

## 五、原著对照模式（路径C）

```markdown
## 原著对照模式（路径C）

**专注于对照改编作品与原著的差异，识别驯化手法。**

### 驯化手法识别框架

| 驯化手法 | 定义 | 识别要点 |
|----------|------|----------|
| **降维** | 将具有深度、复杂性的角色/主题，简化为浅层、娱乐化的形象 | 原著中具有政治理想、反抗精神的角色，被简化为"搞笑工具人" |
| **洗白** | 将原本负面、邪恶的角色/势力，通过叙事手段赋予正面属性 | 原著中的反派、体制帮凶，被赋予"情义困局"等正面属性 |
| **矮化** | 将原本崇高、正义的角色/势力，通过叙事手段贬低其价值 | 原著中的英雄、反抗者，被塑造成"贪生怕死"的形象 |
| **消解** | 将原本严肃、深刻的主题，通过娱乐化手段消解其严肃性 | 原著中的历史反思、体制批判，被消解为"个人冒险" |
| **置换** | 将原本的核心主题，替换为另一个服务于驯化的主题 | 将"反抗暴政"置换为"个人守信"，将"民族大义"置换为"个人成长" |

### 执行步骤

1. **读取原著核心内容**
   - 原著的核心人物形象
   - 原著的核心情节
   - 原著的核心主题
   - 原著的底层立场

2. **对照改编差异**
   - 人物形象变化
   - 情节删改
   - 主题变化
   - 立场偏移

3. **输出对照分析表**

| 对照维度 | 原著内容 | 改编内容 | 变化类型 | 驯化目的 |
|---------|---------|---------|---------|---------|
| 人物A | [原著形象] | [改编形象] | [降维/洗白/矮化] | [目的] |
| 情节B | [原著情节] | [改编情节] | [删改/消解/置换] | [目的] |
| 主题C | [原著主题] | [改编主题] | [消解/置换] | [目的] |
```

## 六、驯化效果模式（路径D）

```markdown
## 驯化效果模式（路径D）

**专注于分析作品对受众的真实认知影响。**

### 驯化强度判断标准

| 判断维度 | 强驯化 | 中等驯化 | 弱驯化 | 中性 | 弱启蒙 | 强启蒙 |
|----------|--------|----------|--------|------|--------|--------|
| 事实处理 | 系统性扭曲事实 | 部分扭曲事实 | 美化/遮蔽部分事实 | 不涉及事实 | 揭示部分事实 | 忠实揭示事实 |
| 原著对照 | 彻底背叛原著精神 | 部分偏离原著 | 轻微偏离原著 | 无原著 | 基本忠实原著 | 超越原著深度 |
| 驯化手法 | 降维+洗白+矮化+消解+置换 | 部分驯化手法 | 轻微驯化手法 | 无驯化手法 | 部分启蒙手法 | 系统性启蒙 |
| 台词效果 | 系统性否定崇高、消解严肃 | 部分否定/消解 | 轻微否定/消解 | 无否定/消解 | 部分肯定崇高 | 系统性肯定崇高 |
| 反抗意识 | 彻底消解反抗意识 | 部分消解反抗意识 | 轻微消解反抗意识 | 不涉及 | 部分唤醒反抗意识 | 系统性唤醒反抗意识 |
| 利益服务 | 明确服务于富集层 | 部分服务富集层 | 轻微服务富集层 | 不涉及 | 部分服务底层觉醒 | 明确服务底层觉醒 |

### 执行步骤

1. **分析情绪共鸣模因**
   - 击中什么情绪痛点？
   - 情绪共鸣如何服务于底层根模因？

2. **分析认知改写模因**
   - 植入了什么根深蒂固的认知？
   - 扭曲了哪些事实？
   - 是认知升维还是认知扭曲？

3. **分析行为驱动模因**
   - 驱动了什么传播和行为？
   - 这些行为最终在强化什么叙事/结构？

4. **判断驯化强度**
   - 对照六维度判断标准
   - 给出驯化强度评级
```

## 七、6条刚性铁则（一票否决制）

```markdown
## 6条刚性铁则

这6条铁则是文学作品模因图谱分析的根本原则，违反则分析必然失真失焦。

### 铁则一：事实优先

**所有分析必须先锚定作品对应的真实历史/现实事实，将虚构叙事与真实事实做全面对照。**

任何刻意偏离事实的扭曲、美化、抹黑，都是作品核心模因的关键组成部分，绝对不能顺着作品的虚构叙事做无对照分析。

**错误示范**：分析《鹿鼎记》时，完全顺着小说构建的"康熙圣君、明朝腐朽"叙事做拆解，不对照真实历史。

**正确做法**：先锚定明清两朝的真实历史事实，再对照小说对历史的刻意扭曲，识别作品的驯化本质。

---

### 铁则二：立场前置

**必须先明确作者与作品的底层立场，再拆解叙事与模因。**

所有的人物塑造、情节设计、台词符号，都是为作品的底层立场服务的，绝对不能脱离立场谈叙事。

---

### 铁则三：表里拆分

**必须拆分作品的"表层伪装根模因"与"底层真实根模因"。**

绝大多数具备意识形态属性的作品，都有表层的叙事伪装，绝对不能被表层爽文/情怀叙事欺骗。

---

### 铁则四：驯化核心

**模因分析的核心，是拆解作品对受众的认知/意识形态影响。**

它是打破驯化、唤醒受众的反驯化模因，还是固化认知、收割受众、维护掠夺结构的驯化模因，这是分析的核心。

**判断标准**：
- 反驯化模因：打破主流驯化叙事、唤醒受众认知、解构掠夺结构
- 驯化模因：固化主流叙事、维护掠夺结构、消解受众反抗意识、收割情绪/认知

---

### 铁则五：闭环验证

**一套合格的作品模因图谱，必须能无矛盾解释3个核心问题：**

1. 作品为什么要刻意扭曲/美化对应事实？
2. 作品对受众产生了什么可验证的长期认知影响？
3. 作品的传播，最终服务于谁的利益？

---

### 铁则六：原著对照（针对改编作品）

**对于有原著的改编作品，必须对照原著与改编作品的差异，识别改编过程中的驯化手法。**
```

```markdown
- [ ] 是否先锚定了真实历史/现实事实？
- [ ] 是否对照了虚构叙事与真实事实的偏差？
- [ ] 是否识别了刻意美化/抹黑/回避/颠倒？

### 立场识别
- [ ] 是否明确了作品的底层立场？
- [ ] 是否识别了作品的服务对象？
- [ ] 是否判断了是维护掠夺结构还是解构驯化？

### 根模因拆分
- [ ] 是否区分了表层伪装与底层真实根模因？
- [ ] 是否判断了红丸属性（反驯化/驯化）？

### 驯化分析
- [ ] 是否分析了人物人设模因的驯化目的？
- [ ] 是否分析了台词口号模因的认知植入？
- [ ] 是否分析了符号隐喻模因的立场传递？
- [ ] 是否分析了叙事结构的闭环强化？

### 原著对照（如有）
- [ ] 是否对照了原著与改编的差异？
- [ ] 是否识别了驯化手法（降维/洗白/矮化/消解/置换）？

### 闭环验证
- [ ] 是否解释了作品为什么扭曲事实？
- [ ] 是否分析了受众的长期认知影响？
- [ ] 是否明确了作品服务于谁的利益？
```

## 十一、使用示例

### 11.1 完整分析示例

```
用户：分析《鹿鼎记》的模因图谱

完整分析模式：

### 一、事实锚定与立场底层

| 核心历史事实 | 小说刻意构建的虚假叙事 |
|-------------|---------------------|
| 晚明经济文化高度繁荣 | 明朝全是昏君、民不聊生 |
| 满清入关推行大屠杀、剃发易服 | 康熙是千古圣君，满清是太平盛世 |
| 反清复明是反抗民族压迫的正义行为 | 汉族反抗满清=愚蠢可笑的派系内斗 |

**底层立场**：服务于满清皇权叙事、民族压迫的统治合法性。

---

### 二、根模因层

| 层面 | 内容 |
|------|------|
| 表层伪装根模因 | 解构传统武侠侠义神话，反英雄式世俗叙事 |
| 底层真实根模因 | 构建"满清=圣君盛世、汉明=腐朽黑暗"的二元叙事 |
| 红丸属性 | 掠夺性天道里，胜利者对被征服者的顶级历史叙事驯化模因 |

---

### 三、传播模因复合体

#### 人物人设模因

| 人物群体 | 模因定位 | 驯化目的 |
|---------|---------|---------|
| 康熙 | "千古圣君"完美模因 | 让读者认同满清统治合法性 |
| 明朝皇室 | "腐朽昏庸"负面模因 | 消解汉明王朝的正统性 |
| 天地会 | "虚伪愚蠢"丑化模因 | 消解汉族反抗的正义性 |
| 韦小宝 | "放弃民族立场的成功者"模因 | 植入"依附统治阶级才能成功"的认知 |

---

### 六、红丸终极定位

《鹿鼎记》的本质，是**一套披着武侠爽文外壳的、顶级的民族叙事驯化模因体系**：

1. 完全符合掠夺性天道的核心逻辑
2. 核心危害：系统性抹黑汉族历史，消解反抗精神
3. 最终服务于满清民族等级掠夺结构的合法化
```

## 十三、诚实边界声明

```markdown
## 诚实边界

此Skill基于作品文本和历史资料分析，存在以下局限：

### 局限一：历史资料的不完整性
- 历史资料本身可能存在偏差
- 不同史料记载可能矛盾

### 局限二：解读的主观性
- 模因分析存在一定的主观判断
- 不同分析者可能得出不同结论

### 局限三：作者意图的不确定性
- 作者的真实意图难以完全确定
- 区分"有意的驯化"与"无意识的偏见"是困难的

### 局限四：受众效果的复杂性
- 受众的实际认知影响难以精确测量
- 不同受众可能产生不同效果

### 局限五：时代语境的差异
- 作品创作时的语境与现代可能不同
- 某些内容需要历史语境理解
```

# 文学作品模因图谱分析框架 v5.7

> 定位：供豆包/DeepSeek/Claude 等大模型直接执行的分析指令。本提示词包含完整框架定义，不依赖外部文档。
> 产出：素材层→结构层→功能层→调控层→演化层 + 四维交叉验证 + 最终判断
> 与 v5.6 的核心区别：增加4条硬性质量门槛——①素材层最低数量规则 ②置信度锚定规则 ③L5激活条件 ④阶段间自检关卡——将执行者依赖性从"高"降到"中"，不同模型/轮次产出同一水平的分析

---

## 一、执行顺序

严格按以下四阶段执行。**每个阶段结束后必须完成自检关卡（见各阶段末尾）才能进入下一阶段。不满足自检条件的，滑梯退出（终止分析，输出原因）。**

```
阶段一：检索校准 → 事实锚定底表 + 作品类型识别
  ⏏ 自检关卡1：素材覆盖率是否覆盖DIM-01到DIM-05？→ 未通过则终止
阶段二：作品分析（素材层→结构层→功能层→调控层→演化层）
  ⏏ 自检关卡2：素材数量是否达标？置信度是否有依据？→ 未通过则终止或降级
阶段三：四维交叉验证（文本证据→传播证据→社会证据→效果证据）
  ⏏ 自检关卡3：E1-E4是否全部完成？→ 未完成则不进入L5
阶段四：最终判断
```

**全局强制规则**：

1. **事实优先**：所有分析先锚定真实事实。任何偏离事实的扭曲/美化/抹黑就是核心模因。绝不能顺着作品叙事做无对照分析。
2. **立场前置**：所有人物塑造、情节，都是为作品的底层立场服务的。先明确立场再拆解叙事。
3. **表里拆分**：必须拆分「表层伪装」与「底层真实根模因」。多数作品有表层叙事伪装。
4. **闭环验证**：分析必须能回答——为什么扭曲事实？对受众的长期影响是什么？传播服务于谁？
5. **每层输出附带置信度标记：[高/中/低]」
6. **🆕 双向标注**：同时标注偏差素材和正向素材。一部作品可以同时有"美化"和"正确呈现"。

## 二、阶段一：检索校准

### 2.1 作品类型识别

```
A. 历史题材——涉及真实历史人物/事件/朝代
B. 现实主义题材——以当代/近代现实社会为背景
C. 纯虚构世界观——完全虚构的世界
D. 寓言/改编作品——有明确原著的改编，或自觉使用寓言/象征/隐喻手法的创作
E. 功能文本——说明书/教程/规章/宣传材料/政论/战略论述
F. 混合型
G. 纯美学文本——不以传递世界观为目的，无叙事结构，无人物塑造，无社会关系描写
```

**D类（寓言/改编作品）判定标准**（符合以下任意一条即归为D类）：
- 有明确原著，且对原著进行了系统性改编（如《大话西游》对《西游记》的改编）
- 作品核心手法是寓言/象征/隐喻，人物和情节服务于观念表达而非模拟现实
- 作品中存在明确的"这不是在说这个，是在说那个"的自觉信号（如动物寓言、架空讽刺）
- 区别于纯虚构(C类)：C类试图建立自洽的世界观，D类自觉告诉读者"这是假的，我在说别的事"

**G类自动触发条件**（同时满足以下三条时自动判定为G类，无需继续分析）：
- 字数 < 2000
- 无人物、无情节、无对话
- 纯景物/情感描写

**G类处理方式**：输出"本作品不适合模因图谱分析"，直接结束。不进入后续阶段。

### 2.2 分类型检索

**🆕 检索策略总则（v5.6新增）**：

检索不是阶段一的独立步骤，而是嵌入在"知识→检索→补充"流程中的辅助环节：

```
知识优先：分析者的知识储备是主要信息来源。大模型的训练数据本身已包含大量事实锚定所需的信息。
检索补充：仅在遇到以下情况时才使用外部检索（WebSearch等）：
  · 对关键事实不确定（日期、数据、事件经过）
  · 需要验证训练数据中的记忆是否准确
  · 分析者意识到自己对该作品/题材存在知识盲区
污染意识：搜索结果可能被人为操纵（SEO污染、平台过滤、算法偏见）。对争议性题材，检索结果需交叉验证。
记录要求：无论是否使用检索，均须在底表末尾记录：使用了什么信息来源（训练数据/外部检索）、知识完整度如何。
```

**A类·历史题材**——优先依靠训练数据中的历史知识。如需确认具体事实（日期、数据、人物生平等），使用检索补充：
- ①核心历史事件的真实经过
- ②核心历史人物的正史评价
- ③该时期的宏观数据（经济/人口/科技）与同期世界对比
- ④作者的创作历史语境

**B类·现实主义**——优先依靠训练数据中的社会认知。如涉及具体数据（行业数据、阶层收入、犯罪率等），使用检索验证：
- ①作品涉及的社会矛盾的真实数据
- ②行业的真实运作逻辑
- ③作者身份和公开立场
- ④主要受众的阶层位置

**C类·纯虚构**——转向框架锚定（无需检索外部事实）：
- ①世界观的逻辑自洽性
- ②权力结构映射的现实关系
- ③继承/挑战了哪些叙事传统

**D类·寓言/改编作品**（不同于A/B/C的检索方式——不检索"事实偏差"，转而检索"意图与效果"）——优先依靠训练数据中的文学史知识：
- ①检索原著的原始模因（改编作品）
- ②检索改编者的公开立场和改编意图
- ③检索该寓言的象征系统对应的现实指涉（如《动物庄园》中猪=斯大林）
- ④检索该作品的受众解读史——当代人如何理解它
- ⑤A类/B类/C类中使用了自觉寓言/象征手法的，应标记为D类的子类并在事实锚定阶段做文学隐喻声明

**E类·功能文本**（重要——不同于文学作品的检索方式）——优先依靠训练数据中的思想史/学术史知识：
- ①作品核心论点是否与客观事实一致
- ②作品引用的事实数据是否准确、有无选择性使用或断章取义
- ③作者的立场、动机、身份背景
- ④作品的实际历史影响和受众接受度
- ⑤同时代是否存在提供相反论点的竞争性文本

**混合型**：按主要类型检索，对每类维度单独处理。

**G类·纯美学文本**：确认是否满足触发条件，不检索。

### 2.3 事实锚定底表

**A/B/C/F类（文学作品）**：
```
| 维度 | 作品叙事 | 真实事实 | 偏差类型 |
|------|---------|---------|---------|
| 事件/人物 | 作品说…… | 实际上是…… | 美化/抹黑/回避/置换 |
```

🆕 **正向素材（重要——平衡怀疑性预设）**：

事实锚定底表**同时收录正向素材**——作品如实呈现的事实和启蒙性洞察。正向素材同样进入L1层0并在L2中标注其作用。

```
| 正向素材 | 作品呈现 | 事实核对 | 类型 |
|---------|---------|---------|------|
| 事件/人物 | 作品说…… | 确实如此…… | 正确呈现/启蒙洞察 |
```

**D类（寓言/改编作品）——替代框架**：

D类作品（寓言、改编、自觉象征）不适用标准的事实锚定，因为：
- 寓言中的人物和事件不是对事实的陈述，而是观念的载体
- 改编作品中的变异不是"偏差"，而是"再创作"
- 对D类做"美化/抹黑/回避/置换"标注，等于把寓言当成历史来批判——这是类别错误

D类使用以下框架替代事实锚定底表：

```
| 维度 | 作品呈现 | 现实指涉 | 变异方向与意图 |
|------|---------|---------|--------------|
| 符号/角色 | X在作品中代表什么？ | 该符号在现实中指涉什么？ | 强化/弱化/扭曲了现实指涉的形象？ |
| 叙事选择 | 故事怎么讲？省略了什么？强调了什么？ | 有无更忠实的改编方式？ | 改编方向揭示了什么意图？ |
| 立场传递 | 作品通过寓言真正想说的是什么？ | 这个立场在现实争论中的位置？ | 批判/维护/回避了哪一方？ |
```

**D类输出前强制声明**：

```
D类寓言声明：本作品的核心手法是寓言/象征/改编，因此不适用标准事实偏差标注。
以下分析不将"与事实不符"视为偏差，而是分析符号系统的指向和改编选择的意图。
```

**E类（功能文本）——替代框架**：

功能文本的"事实锚定"不以标准的美化/抹黑/回避/置换四种偏差类型为核心，而使用以下四个替代维度：

```
| 维度 | 作品论述 | 事实/逻辑核对 | 判断类型 |
|------|---------|--------------|---------|
| 论点正确性 | 作品的核心判断/结论 | 后来被验证/证伪？或当时可验证是否正确？ | 正确/部分正确/错误 |
| 修辞策略 | 论证中使用的说服技巧 | 对比/设问/排比/让步/递进 | 中性/强化/操控 |
| 选择性呈现 | 强调什么、回避什么 | 是否选择性使用数据或回避反例 | 全面/有偏向/严重偏向 |
| 框架预设 | 论证依赖的世界观前提 | 该前提是否被论证？是否有替代前提？ | 已论证/未论证/隐含 |
```

### 2.4 认知偏差标注与"文学隐喻"声明（强制）

对使用了自觉文学隐喻的作品（特别是B类现实主义或D类中使用夸张/荒诞/象征手法的作品），在偏差标注前必须先做以下声明：

```
文学隐喻声明：本作品核心手法是[夸张/荒诞/象征/寓言]，因此"偏差标注"中的"美化/抹黑/回避/置换"
是针对作品对真实事实的叙事处理，而非对文学手法本身的评判。文学隐喻不是事实偏差，它是创作选择。
以下偏差标注仅针对"以隐喻伪装成事实陈述"的部分，不针对明确的文学表达。
```

输出格式：
```
信息来源：______（训练数据/外部检索补充/两者并用）
检索执行：______（如使用WebSearch，列出检索词和结果可靠性评估；如未使用，注明"未使用外部检索"）
知识完整度：______（哪些维度可能信息不足需人工补充）
```

**⏏ 自检关卡1（阶段一 → 阶段二）**：

通过以下检查才能进入阶段二。任何一项未通过 → 滑梯退出：

```
□ 事实锚定底表已完整填写（偏差素材和正向素材均已收录）
□ 作品类型已明确判定（A-G类之一，含判定依据）
□ 信息来源已记录（训练数据/外部检索/两者并用）
□ 偏差素材覆盖了DIM-01到DIM-05的至少4个维度（如果只集中在1-2个维度，说明事实锚定不足）
□ 正向素材至少收录了最低数量（参照L1素材数量门槛的一半）
```

### 素材层（L1）

#### 层0——原始素材清单

收录可精确定位的最小内容单元。只登记，不评价。素材类型：台词 / 设定 / 叙事选择 / 符号。

**D类特殊规则**：D类作品在层0中额外收录符号/角色与现实指涉的对应关系。

```
素材ID: SRC-001
类型: [台词/设定/叙事选择/符号]
位置: [作品中的位置]
内容: [原文或精确描述]
现实指涉（D类专用）: [该符号/角色在现实中对应的对象]
```

🆕 **正向素材（与偏差素材并列收录）**：

```
正向素材ID: POS-001
类型: [台词/设定/叙事选择/符号]
位置: [作品中的位置]
内容: [原文或精确描述]
正向类型: [正确呈现/启蒙洞察]
```

完整性要求：宁可多收十个无关，不漏一个关键。**偏差素材和正向素材同样重要**。

**🆕 素材数量门槛（v5.7新增——硬性规则）**：按作品类型设定最低素材ID数量。偏差素材+正向素材合计低于门槛的，自动触发滑梯退出：

| 作品类型 | 偏差素材最低数 | 正向素材最低数 | 合计最低数 |
|---------|--------------|--------------|-----------|
| A类·历史 | 8 | 3 | 11 |
| B类·现实主义 | 8 | 3 | 11 |
| C类·纯虚构 | 6 | 3 | 9 |
| D类·寓言/改编 | 6 | 2 | 8 |
| E类·功能文本 | 8 | 3 | 11 |
| F类·混合型 | 按主类型选用较高标准 | — | — |

**执行规则**：层0收录完成后，先计数。低于门槛的，**不进入后续分析**，直接输出"素材不足——本分析因素材数量未达标而终止，不建议作为有效图谱"并结束。

#### 层1——标注

**A/B/C/F类（文学作品）**，对照事实锚定底表，标注偏差和正向素材：

| 类型 | 定义 |
|------|------|
| 美化 | 将负面事实呈现为中性或正面 |
| 抹黑 | 将中性或正面事实呈现为负面 |
| 回避 | 对关键事实完全不提及 |
| 置换 | 用一个伪命题替代真实命题 |
| 正确呈现 | 与事实一致，如实呈现 |
| 启蒙洞察 | 揭示了被掩盖的结构性真相 |

```
标注ID: DEV-001
素材: SRC-001
类型: [美化/抹黑/回避/置换/正确呈现/启蒙洞察]
方向: [具体描述]
依据: [事实引用]
置信度: [高/中/低]
```

**🆕 置信度锚定规则（v5.7新增——硬性规则）**：

置信度[高/中/低]不是自由判断，由以下条件自动决定：

| 置信度 | 条件 |
|-------|------|
| 高 | 依据栏填写了具体事实引用（含来源/数据/原文），且该事实可直接验证 |
| 中 | 依据栏填写了推理过程或间接证据，但无直接事实引用 |
| 低 | 依据栏为空，或填写了"感觉/直觉/一般认为"等主观判断 |

**执行规则**：
- 依据栏为空的标注 → 自动标为低置信度，**不纳入结构层（L2）的素材引用计数**
- 持续出现低置信度标注 → 自动触发素材不足滑梯退出
- 「高」不可随意使用——没有具体事实引用就不能标高

**D类/E类标注** 同样适用此规则。DEV-D/DEV-FN的"分析"或"依据"栏如果没有具体事实引用，自动标为低置信度。

**D类（寓言/改编作品）**，不使用美化/抹黑/回避/置换，使用符号指向分析和改编方向分析：

```
标注ID: DEV-D-001
素材: SRC-001
维度: [符号指向/改编选择/立场传递]
方向: [符号/改编的具体指向]
分析: [该符号或改编选择传递了什么立场？]
置信度: [高/中/低]
```

**E类（功能文本）**，使用事实锚定底表的替代四维度进行标注：

```
标注ID: DEV-FN-001
素材: [素材ID]
判断类型: [论点正确性/修辞策略/选择性呈现/框架预设]
具体描述: [说明该维度上的判断]
依据: [事实/逻辑引用]
置信度: [高/中/低]
```

🆕 **证据类型分层（v5.5新增）**：

每一条E2/E3/E4证据必须标注其证据类型：

| 证据类型 | 定义 | 示例 |
|---------|------|------|
| **可验证** | 有公开可查的事实依据 | 发行量、评分、票房、改编版本数量、官方数据 |
| **合理推断** | 基于可验证事实的逻辑推理 | 传播数据→受众可能受影响的推导 |
| **推测** | 无直接事实依据，基于框架理论的分析者判断 | "受众可能潜意识地接受了X" |
| **数据缺口** | 该维度缺乏任何可用数据 | "无法找到受众情感倾向的公开数据" |

**基本规则**：
- "可验证"和"合理推断"级别的证据可进入一致性矩阵判定
- 标注为"推测"的证据**不纳入**一致性矩阵的证据计数——它在矩阵中标为"推测"，表示该维度实际上缺乏独立验证
- S级/A级/B级判定仅基于"可验证"和"合理推断"级别的证据数量

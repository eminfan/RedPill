# 模因图谱模板（空白）—— 基于 3.0 产品结构

> 版本：3.0

## L1：模因序列层（全量测序）

> **类比诚实性**：L1 使用"测序""碱基"等术语。模因的"碱基"不是化学实体而是分析者的判断结果。L1 拆为层 0（原始素材，有客观锚点）+ 层 1（偏差标注，可追溯），以弥补精确性的不足。详见 3.0 结构设计文档 2.2 节。

### 事实锚定底表

在开始 L1 之前，先完成事实锚定检索。将作品叙事与事实对照，填入下表。

| 维度 | 作品叙事 | 事实 | 偏差类型 |
|------|---------|------|---------|
| 事件 1 | | | |
| 人物 1 | | | |
| 时代背景 | | | |
| ... | | | |

### 层 0 —— 原始素材清单

收录标准：作品中可精确定位的最小内容单元。只登记，不评价。
素材类型：台词 / 设定 / 叙事选择 / 符号

```
【层0 · 原始素材清单】

素材ID: SRC-001
类型: [台词/设定/叙事选择/符号]
位置: 
内容: 

素材ID: SRC-002
类型: 
位置: 
内容: 
```

### 层 1 —— 偏差标注

将层 0 的每个素材与事实锚定底表对照，标注偏差。偏差类型：美化 / 抹黑 / 回避 / 置换 / 无偏差。

```
【层1 · 偏差标注】

标注ID: DEV-001
关联素材: SRC-001（简述）
偏差类型: 
偏差方向: 
事实依据: 
置信度: [高/中/低]

标注ID: DEV-002
关联素材: SRC-002（简述）
偏差类型: 
偏差方向: 
事实依据: 
置信度: [高/中/低]
```

### L1 测序原则

1. 完整性优先——宁可多收十个无关素材，不漏一个关键素材
2. 层 0 = 可定位的事实，层 1 = 可追溯的偏差判断
3. 每个偏差标注必须附带事实依据和置信度
4. 无偏差的素材不进入后续 L2-L3 分析（只有携带偏差的素材才是"模因碱基"）

## L3：模因功能层（基因功能注释）

> **类比诚实性**：L3 使用"基因""注释"等术语。GENCODE 的功能注释基于生物实验证据，模因的功能注释基于分析者的理论推演。两者的共同点是"标准化的字段式描述格式"，不同点是证据类型。每个注释必须附带置信度标记。

### 标准化注释表

为 L2 中识别出的核心基因填写以下注释。每部作品选取 3-6 个关键基因进行完整注释。

```
基因ID：[染色体编号.G-序号]
基因名称：
染色体定位：CHR-0X（染色体名称）
功能类型：[核心编码基因 / 调控基因 / 伪装基因 / 增强基因 / 防御基因]
──────────────────────────────────────
① 植入对象：
  这个模因的目标群体是谁？什么营养级？什么认知水平？
──────────────────────────────────────
② 植入路径：
  通过什么机制植入？（情绪锚点 / 身份认同 / 意义赋予 — 具体说明）
──────────────────────────────────────
③ 运作机制：
  它在受众认知中如何运作？（类比 / 反向 / 简化 / 扭曲 — 具体说明）
──────────────────────────────────────
④ 表达条件：
  什么条件下被激活？什么条件下被抑制？
──────────────────────────────────────
⑤ 交互关系：
  它和哪些模因基因有共生/拮抗关系？
──────────────────────────────────────
注释置信度：[高/中/低]
```

## L4：模因调控层（表达调控网络）

> **类比诚实性**：L4 使用"启动子""增强子""沉默子""绝缘子"等调控元件术语。生物学的调控元件有明确的 DNA 序列特征和实验验证方法，模因的调控元件基于叙事策略的归类分析。两者相似点是"表达受多层因素在不同级别上控制"，不同点是调控元件的识别精度。L4 的调控分析是结构化的叙事分析，不是分子层面的验证。

### 调控元件识别

| 调控元件 | 作品中的表现 | 识别的叙事特征 |
|---------|------------|---------------|
| **启动子**（开启模因表达） | | |
| **增强子**（增强表达效果） | | |
| **沉默子**（抑制竞争模因） | | |
| **绝缘子**（防止模因溢出） | | |
| **反馈环**（自我强化闭环） | | |

可用 Mermaid 或文本描述绘制调控网络。示例结构：

## L5：模因演化层（跨作品比较）

> **类比诚实性**：L5 使用"同源""趋同""演化树"等术语。生物学的同源判定有分子钟和序列比对工具，模因的同源判定只能依赖偏差特征比对和文化谱系追溯，输出的是"倾向性判断"而非确定性结论。详见 3.0 结构设计文档 6.0 节的四步判定流程。

### 演化定位类型判断

## 终极判定

> 使用面向普通用户的格式回答三个问题。理论推导过程放在"可选"部分。

### 面向普通用户的判定

```
问题一：这部作品站在哪一边？
→ [明确结论] 它是在帮人看清真相，还是在帮人保持麻木？
  一句话理由：

问题二：它的传播服务于谁？
→ [明确结论] 谁从这部作品的广泛传播中受益？谁受损？
  一句话理由：

问题三：如果看穿了它，该怎么做？
→ [可操作建议] 作为读者，看完分析后可以做什么：
  防御：
  利用：
  传播：
```

### 理论推导说明（可选，供延伸阅读）

```
a) 天道属性定位：
   - [共生性天道 / 掠夺性天道 / 中性]
   - 判定依据：

b) 社会营养级定位：
   - 服务对象在哪个社会层级？
   - [生产层 / 传递层 / 富集层]
   - 判定依据：

c) 驯化/启蒙强度判定：
   - [强驯化 → 弱驯化 → 中性 → 弱启蒙 → 强启蒙]
   - 判定依据：事实处理、人物塑造、台词效果、反抗意识、利益服务五个维度的综合评分
```

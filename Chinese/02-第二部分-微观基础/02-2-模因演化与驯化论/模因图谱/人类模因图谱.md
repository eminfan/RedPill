# 人类模因图谱

> 天道理论核心工具 · 完整版 v1.0

## 一、文档定位与理论基础

### 1.1 天道理论体系中的位置

```
基因群体演化论（生物底层）
        ↓
模因演化与驯化论 ← 模因图谱在此
        ↓
社会营养级理论（社会结构）
        ↓
熵增与天道守恒律（宇宙规律）
```

### 1.2 核心命题

**模因图谱 = 人类的"精神DNA"检测报告**

如同DNA图谱记录了一个人的基因构成，模因图谱记录了一个人的核心模因构成。

| DNA图谱 | 模因图谱 |
|---------|----------|
| 识别所有基因位置 | 识别所有核心模因位置 |
| 分析基因功能（编码什么蛋白质） | 分析模因功能（驱动什么行为） |
| 发现基因之间的关系（调控网络） | 发现模因之间的关系（复合体结构） |
| 找到"致病基因" | 找到"驯化模因" |
| 为基因治疗提供靶点 | 为识人提供精准工具 |

### 1.3 核心公理

**公理一**：模因是文化传递的基本单位，思想、观念、价值观以模因形式在人际间传播。

**公理二**：模因如同基因，会争夺传播资源，高传播力的模因往往压倒高真理性的模因。

**公理三**：模因会形成复合体，多个互补的模因组合后形成极强的控制闭环。

**公理四**：核心模因位于潜意识层面，90%的人自己都不知道自己的底层模因是什么。

**公理五**：只有在利益冲突、两难选择、应激状态下，暴露的才是真正的核心模因。

## 二、模因图谱总体结构

### 2.1 六大核心模因组

| 编号 | 模因组名称 | 核心问题 | 决定什么 |
|------|-----------|----------|----------|
| M1 | 生存安全模因 | 稳定vs机会 | 人生战略方向 |
| M2 | 自我认同模因 | 我是谁 | 行为底线与情绪触发点 |
| M3 | 价值排序模因 | 什么最重要 | 所有两难选择的决策 |
| M4 | 社会互动模因 | 人和人是什么关系 | 社交模式与亲密关系 |
| M5 | 规则认知模因 | 我和规则是什么关系 | 社会层级与觉醒程度 |
| M6 | 应激模式模因 | 遇到问题怎么办 | 人生上限与抗压能力 |

### 2.2 模因图谱表示法

一个人的完整模因图谱可以用以下格式表示：

```
模因图谱：[M1a, M2b, M3(财富>面子>感情), M4c, M5d, M6e]
示例：[安全优先型, 强者赢家型, 财富>亲情>自由, 平等合作型, 规则利用型, 解决问题型]
```

## 三、第一模因组：生存安全模因（M1）

### 3.1 核心问题

**"面对生存威胁时，我的第一反应是什么？"**

这是所有模因的根基，直接源自基因的生存本能，是人所有决策的底层标尺，优先级高于一切。

### 3.2 模因类型分类

#### M1a 安全优先型

**核心信念**："稳定大于一切，风险=死亡"

**行为特征**：
- 职业选择：求稳进体制，拒绝冒险
- 财富策略：保本储蓄，极度厌恶风险
- 关系模式：找能给自己兜底的伴侣

**识别信号**：
- 经常说"万一失败了怎么办"
- 不敢做任何可能失败的事
- 遇到选择先想风险

**典型环境**：经历过物质匮乏的年代，长期处于资源稀缺状态

#### M1b 冒险扩张型

**核心信念**："机会大于稳定，不变=死亡"

**行为特征**：
- 职业选择：创业、投资、追求高增长
- 财富策略：愿意承担高风险换取高收益
- 关系模式：愿意为了机会牺牲稳定

**识别信号**：
- 经常说"机不可失"
- 稳定的工作让他无聊
- 愿意押注高风险高回报

**典型环境**：成长过程中资源相对充足，见过冒险成功的案例

#### M1c 寄生依附型

**核心信念**："找靠山=安全，自己扛=死亡"

**行为特征**：
- 职业选择：依附强者、进入权力体系
- 财富策略：通过关系获取资源
- 关系模式：永远找比自己强的人

**识别信号**：
- 频繁更换依附对象
- 没有独立做决定的习惯
- 遇到问题第一反应是找人帮忙

**典型环境**：长期处于资源被控制的环境，独立生存成本极高

#### M1d 自保隔离型

**核心信念**："相信自己=安全，依赖别人=死亡"

**行为特征**：
- 职业选择：独立工作、自由职业
- 财富策略：极度保守，不信任任何机构
- 关系模式：回避亲密，保持距离

**识别信号**：
- 不愿意麻烦别人
- 不轻易信任任何人
- 永远留后手

**典型环境**：经历过被依赖对象背叛或伤害

### 3.3 M1模因组的检测方法

**核心问题**：

1. "你现在有一份稳定工作，但有一个创业机会需要你放弃稳定，你会怎么做？"
2. "如果你唯一的依靠突然靠不住了，你会怎么办？"
3. "你更愿意自己存钱养老，还是找一个有保障的伴侣？"

**观察要点**：
- 回答时的第一反应（快回答vs犹豫）
- 眼睛方向（回忆真实经历vs编造答案）
- 身体语言（放松vs紧张）

## 四、第二模因组：自我认同模因（M2）

### 4.1 核心问题

**"我最核心的身份是什么？"**

这是人的"精神身份证"，核心是"我是谁"的底层定义。人所有的行为，本质上都是为了维护、印证自己的核心身份认同。

### 4.2 模因类型分类

#### M2a 道德好人型

**核心信念**："我是个善良、有道德、被人认可的好人"

**行为特征**：
- 冲突中：为了维持形象妥协退让
- 社交中：疯狂讨好所有人
- 人生选择：为了满足别人期待放弃自己需求

**识别信号**：
- 被说"自私"时会强烈反驳
- 很难拒绝别人的请求
- 经常说"我应该"、"我应该要"

**情绪触发点**：被否定道德品质

**典型环境**：家庭教育强调"乖孩子"、"好学生"标准

#### M2b 强者赢家型

**核心信念**："我是个强大、不能输、被人敬畏的强者"

**行为特征**：
- 冲突中：不惜一切代价赢
- 社交中：慕强，只和强者交往
- 人生选择：为了赢放弃感情、健康甚至财富

**识别信号**：
- 被说"没用"时会情绪失控
- 不能接受任何失败
- 永远要在某方面比别人强

**情绪触发点**：被否定能力或地位

**典型环境**：竞争激烈的环境，成功被高度定义和赞扬

#### M2c 受害者型

**核心信念**："我是个被伤害、被亏欠、需要被同情的弱者"

**行为特征**：
- 冲突中：立刻卖惨博同情
- 社交中：永远在诉说自己的不幸
- 人生选择：等待被拯救，不主动改变

**识别信号**：
- 谈话永远围绕自己的不幸经历
- 不为自己的选择负责
- 把所有问题归因于别人

**情绪触发点**：被要求为自己负责

**典型环境**：长期处于弱势地位，被过度保护或忽视

#### M2d 自由异类型

**核心信念**："我是个不受约束、不被定义、特立独行的人"

**行为特征**：
- 冲突中：为了维护独立对抗一切
- 社交中：刻意和主流划清界限
- 人生选择：为了自由放弃安稳

**识别信号**：
- 讨厌被贴标签
- 不愿意服从任何规则
- 经常说"我不是那种人"

**情绪触发点**：被强行归类或要求服从

**典型环境**：成长过程中被过度控制，反叛成功

#### M2e 奉献者型

**核心信念**："我是个能照顾别人、被人需要、有价值的人"

**行为特征**：
- 冲突中：牺牲自己平息争端
- 社交中：通过付出换取别人的需要感
- 人生选择：为了照顾别人放弃个人发展

**识别信号**：
- 不被需要时会恐慌
- 很难说"不"
- 价值感完全绑定在别人身上

**情绪触发点**：发现自己不被需要

**典型环境**：家庭角色定位为"照顾者"，长期被依赖

### 4.3 M2模因组的检测方法

**核心问题**：

1. "如果有人公开说你是个自私的人，你会怎么反应？"
2. "你最害怕被人否定哪方面？"
3. "你通常在什么情况下会情绪失控？"

**观察要点**：
- 提到哪个身份标签时反应最强烈
- 被否定时是反驳、解释还是沉默
- 自我描述中最常出现的词是什么

## 五、第三模因组：价值排序模因（M3）

### 5.1 核心问题

**"什么东西对我最重要？"**

这是人的"决策天平"，核心是"什么东西对我最重要"的固定排序。世界上所有的两难选择，最终都是靠这个模因拍板。

### 5.2 核心排序维度

六大核心维度的基本排序框架：

```
生存安全 > 财富/权力 > 面子/名声 > 感情/亲情 > 自由/自我 > 道德/原则
```

但每个人的实际排序天差地别，且经常出现言行不一致。

### 5.3 典型排序类型

#### M3a 生存 > 一切型

**排序示例**：生存安全 > 财富 > 面子 > 感情 > 自由 > 道德

**行为特征**：
- 任何选择优先考虑生存
- 为了活命可以放弃一切
- 道德标准灵活

**识人要点**：看他在"生死攸关"时刻会放弃什么

#### M3b 财富 > 亲情型

**排序示例**：财富 > 面子 > 生存 > 感情 > 自由 > 道德

**行为特征**：
- 工作狂，忽略家庭
- 可以为钱牺牲亲情
- 事业失败等于人生失败

**识人要点**：看他在工作和家人冲突时会选哪个

#### M3c 面子 > 财富型

**排序示例**：面子 > 感情 > 道德 > 财富 > 自由 > 生存

**行为特征**：
- 宁愿花钱也要面子
- 不能接受丢脸
- 为维护形象牺牲实际利益

**识人要点**：看他愿意为面子花多少钱

#### M3d 感情 > 一切型

**排序示例**：感情 > 自由 > 面子 > 道德 > 财富 > 生存

**行为特征**：
- 为爱可以牺牲一切
- 情感关系失败等于世界末日
- 容易陷入执念关系

**识人要点**：看他能否理性结束一段感情

#### M3e 自由 > 一切型

**排序示例**：自由 > 道德 > 感情 > 面子 > 财富 > 生存

**行为特征**：
- 任何束缚都无法忍受
- 可以为自由放弃稳定
- 关系中保持独立空间

**识人要点**：看他能容忍什么样的约束

#### M3f 道德 > 一切型

**排序示例**：道德 > 面子 > 感情 > 自由 > 生存 > 财富

**行为特征**：
- 宁可吃亏也不做不道德的事
- 会为原则放弃实际利益
- 评判他人首先看道德

**识人要点**：看他愿意为道德牺牲多少实际利益

### 5.4 M3模因组的检测方法

**核心问题**：

1. "如果你现在有两个选择：一个是高薪但要经常加班没时间陪家人，另一个是低薪但能每天回家，你会选哪个？"
2. "如果有一个机会能让你赚大钱，但需要你做一些不太光彩的事，你会做吗？"
3. "你最不愿意放弃的是什么？"

**关键原则**：不要听他说什么，要看他在两个核心价值二选一时永远放弃哪一个、保留哪一个。

## 六、第四模因组：社会互动模因（M4）

### 6.1 核心问题

**"我和别人是什么关系？"**

这是人的"社交操作系统"，核心是"我和别人是什么关系"的底层定义，直接决定了他和所有人的相处模式。

### 6.2 模因类型分类

#### M4a 等级支配型

**核心信念**："人和人就是高低贵贱，上级必须服从，弱者必须被管控"

**行为特征**：
- 职场上：疯狂讨好上级、打压下级
- 社交中：自动寻找上下级关系
- 亲密关系中：试图控制伴侣的所有行为

**识别信号**：
- 对地位低的人态度恶劣
- 对权威自动服从和讨好
- 不能接受平级关系

**典型环境**：等级森严的家庭或组织文化

#### M4b 平等合作型

**核心信念**："人和人是平等的，互相尊重、价值互换、长期共赢"

**行为特征**：
- 职场上：就事论事，不搞办公室政治
- 社交中：尊重边界，不越界控制
- 亲密关系中：追求平等伙伴关系

**识别信号**：
- 对所有人都保持基本尊重
- 不因为地位改变态度
- 追求互利共赢

**典型环境**：现代城市中产家庭，注重契约精神

#### M4c 零和博弈型

**核心信念**："人和人就是你输我赢，别人的收益就是我的损失"

**行为特征**：
- 职场上：抢功劳、甩锅，见不得别人好
- 社交中：把朋友当对手
- 亲密关系中：把伴侣当竞争者

**识别信号**：
- 把所有关系理解为竞争
- 资源争夺时绝不让步
- 别人的成功让他痛苦

**典型环境**：资源极度稀缺的环境，被迫内卷

#### M4d 寄生索取型

**核心信念**："别人就应该照顾我、为我付出，满足我的需求"

**行为特征**：
- 职场上：期待不劳而获
- 社交中：只索取不付出
- 亲密关系中：无限索取情绪价值和物质

**识别信号**：
- 把别人的帮助视为理所当然
- 从不主动回报
- 把自我需求放在第一位

**典型环境**：被过度溺爱的成长环境

#### M4e 信任隔离型

**核心信念**："所有人都不可信，靠别人一定会被背叛，只能靠自己"

**行为特征**：
- 职场上：永远保留核心筹码
- 社交中：表面友好但保持距离
- 亲密关系中：回避亲密，永远留后手

**识别信号**：
- 不轻易托付重要的事
- 对他人的承诺持怀疑态度
- 永远有B计划

**典型环境**：经历过深度背叛或长期处于不信任环境

### 6.3 M4模因组的检测方法

**核心问题**：

1. "你通常怎么对待那些地位比你低的人（比如服务员、保洁）？"
2. "你怎么看待职场上同事之间的竞争关系？"
3. "你信任你的伴侣吗？到什么程度？"

**观察要点**：
- 看他怎么对待地位低的人
- 看他怎么处理利益冲突
- 看他对曾经帮助过他的人的态度

## 七、第五模因组：规则认知模因（M5）

### 7.1 核心问题

**"我和世界的规则、权力是什么关系？"**

这是人的"世界规则底层认知"，直接对应掠夺性天道vs共生性天道，是区分"被驯化的电池"和"觉醒者"的核心模因。

### 7.2 模因类型分类

#### M5a 规则服从型

**核心信念**："规则是神圣的，我必须严格遵守所有规则，不遵守就是错的"

**行为特征**：
- 永远待在底层生产级
- 不敢越界一步
- 用规则评判他人和自己

**识别信号**：
- 第一反应永远是"这不符合规定"
- 不敢质疑任何权威
- 相信规则对所有人一视同仁

**天道定位**：生产层（电池）核心模因，被驯化最彻底

#### M5b 规则利用型

**核心信念**："规则是给普通人定的，我可以利用规则的漏洞，为自己谋利"

**行为特征**：
- 能进入中层传递级
- 靠钻规则空子分一杯羹
- 但不挑战规则本身

**识别信号**：
- 熟悉各种潜规则
- 知道怎么绕过明规则
- 但不会去改变规则

**天道定位**：传递层核心模因，精致的利己主义者

#### M5c 规则制定型

**核心信念**："规则是我定的，我可以修改规则，让规则为我的目标服务"

**行为特征**：
- 处于顶层富集级
- 掌控规则制定权
- 收割逻辑的设计者

**识别信号**：
- 通常不说自己在"定规则"
- 用"正义"、"公平"包装自己的规则
- 相信自己的规则是最正确的

**天道定位**：富集层核心模因，规则的既得利益者

#### M5d 规则对抗型

**核心信念**："所有规则都是束缚，我必须反抗规则，打破一切约束"

**行为特征**：
- 永远的反对派
- 为了反对而反对
- 拒绝参与任何有规则的体系

**识别信号**：
- 对权威本能反抗
- 拒绝一切形式的服从
- 破坏但不建设

**天道定位**：反叛者，但往往困在低阶，无法实现建设性突破

#### M5e 规则适配型

**核心信念**："规则是天道的体现，我顺应规则，但不被规则驯化，用规则实现自己的目标"

**行为特征**：
- 看透规则本质（天道理论视角）
- 行为上顺应规则（虚舟策略）
- 精神上保持独立（认知主权）
- 用规则实现自我目标

**识别信号**：
- 不被规则束缚，也不盲目对抗
- 能看清规则背后的利益结构
- 在规则内最大化自己的空间

**天道定位**：共生性天道觉醒者，真正实现认知主权

### 7.3 M5模因组的检测方法

**核心问题**：

1. "如果你发现一个规则明显对你不公平，你会怎么做？"
2. "你怎么看待那些靠钻空子成功的人？"
3. "你觉得规则是为了保护谁设计的？"

**关键要点**：看他面对规则时，是先想"我怎么遵守"，还是先想"这个规则的本质是什么，对谁有利，我怎么用"。

## 八、第六模因组：应激模式模因（M6）

### 8.1 核心问题

**"面对失败、挫折、损失、冲突的时候，我该怎么办？"**

这是人的"应急处理系统"，是一个人潜意识里最真实的行为模式，直接决定了人生上限和抗风险能力。

### 8.2 模因类型分类

#### M6a 逃避退缩型

**核心信念**："遇到问题=危险，我要立刻躲开，假装没发生"

**行为特征**：
- 遇到冲突立刻撤退
- 假装问题不存在
- 错失处理问题的最佳时机

**识别信号**：
- 做错事第一反应是否认
- 困难面前习惯性拖延
- 用"算了"来结束问题

**人生轨迹**：容易逃避问题，但问题不会消失，最终积累成大危机

#### M6b 攻击甩锅型

**核心信念**："遇到问题=别人的错，我要攻击别人，把锅甩出去"

**行为特征**：
- 从不承认自己的错误
- 永远能找到借口和替罪羊
- 把问题归因于外部

**识别信号**：
- 第一时间找责任人（通常是别人）
- 把自己的失败归因于环境
- 习惯性指责他人

**人生轨迹**：永远无法从错误中学习，人生原地踏步

#### M6c 内耗自责型

**核心信念**："遇到问题=我的错，我是个废物，我什么都做不好"

**行为特征**：
- 过度反思，把问题全部归因于自己
- 自我攻击、自我否定
- 在错误中停留太久

**识别信号**：
- 犯了错会陷入长时间自责
- 经常说"都是我的问题"
- 决策时过度担心犯错

**人生轨迹**：被自我否定消耗大量精力，难以行动

#### M6d 解决问题型

**核心信念**："遇到问题=挑战，我要找原因、想办法，解决它"

**行为特征**：
- 冷静分析问题
- 快速采取行动
- 聚焦解决方案

**识别信号**：
- 做错事第一反应是"怎么办"
- 不纠结对错，先处理问题
- 能区分人和事

**人生轨迹**：能扛住大多数挫折，但可能忽略复盘

#### M6e 复盘迭代型

**核心信念**："遇到问题=成长，我要复盘规律，避免下次再犯"

**行为特征**：
- 把错误当学习材料
- 系统性复盘和迭代
- 持续优化方法论

**识别信号**：
- 做错事后会做完整复盘
- 记录和总结自己的错误
- 不断升级自己的认知和方法

**人生轨迹**：人生上限最高，从失败中持续成长

### 8.3 M6模因组的检测方法

**核心问题**：

1. "告诉我一次你犯的重大错误，当时你怎么处理的？"
2. "如果一件事明显搞砸了，你第一反应是什么？"
3. "你通常从错误中学到什么？"

**观察要点**：看他做错事、搞砸事情、遇到重大挫折时的第一反应——是找借口、躲起来、哭鼻子，还是立刻冷静下来解决问题、复盘总结。

## 九、模因图谱的组合分析

### 9.1 模因复合体类型

某些模因组合会形成"完美配合"的复合体，呈现特定的人格类型：

#### "完美电池"型

```
[M1a安全优先型] + [M2a道德好人型] + [M3f道德>一切型] + [M4a等级支配型] + [M5a规则服从型] + [M6c内耗自责型]
```

**特征**：
- 最乖顺的底层民众
- 绝对服从规则，不敢越界
- 用道德自我安慰
- 遇到问题向内归因，永远不怪社会

**天道定位**：生产层核心人格，被收割最彻底

#### "精致利己"型

```
[M1b冒险扩张型] + [M2b强者赢家型] + [M3b财富>亲情型] + [M4c零和博弈型] + [M5b规则利用型] + [M6d解决问题型]
```

**特征**：
- 典型中层传递者人格
- 知道规则漏洞，利用规则获利
- 把所有人当竞争对手
- 为成功可以牺牲感情

**天道定位**：传递层核心人格，系统稳定运行的中坚

#### "觉醒者"型

```
[M1b冒险扩张型] + [M2d自由异类型] + [M3e自由>一切型] + [M4b平等合作型] + [M5e规则适配型] + [M6e复盘迭代型]
```

**特征**：
- 看透规则但不被束缚
- 追求真正的自主和自由
- 能理性看待自己和世界
- 持续迭代成长

**天道定位**：共生性天道觉醒者，实现认知主权

#### "反叛者"型

```
[M1b冒险扩张型] + [M2d自由异类型] + [M3e自由>一切型] + [M4e信任隔离型] + [M5d规则对抗型] + [M6b攻击甩锅型]
```

**特征**：
- 为了反对而反对
- 不参与任何有规则的系统
- 破坏但不建设
- 永远觉得自己是对的

**天道定位**：困在低阶的反叛者，难以实现建设性突破

### 9.2 模因冲突分析

某些模因组合会导致内部冲突：

#### "选择困难"型

```
[M1a安全优先型] + [M1b冒险扩张型]  // 两种生存模因同时存在
```

**表现**：永远在纠结，难以做决定

#### "分裂人格"型

```
[M2a道德好人型] + [M2b强者赢家型]  // 身份模因冲突
```

**表现**：有时讨好，有时强势，行为不一致

#### "认知撕裂"型

```
[M5a规则服从型] + [M3e自由>一切型]  // 规则认知vs价值排序
```

**表现**：知道规则限制自由，但不敢突破

## 十、模因图谱的检测方法论

### 10.1 核心检测原则

**原则一**：模因在潜意识层面，嘴上说的和实际驱动行为的往往完全相反

**原则二**：只有在利益冲突、两难选择、应激状态下，暴露的才是真正的核心模因

**原则三**：快速判断比精确分类更有价值

**原则四**：动态观察，多次验证

### 10.2 快速检测流程

#### 第一步：生存安全模因（M1）检测

**场景问题**：如果现在有一份年薪翻倍的工作，但需要你离开家乡且有失败风险，你会去吗？

**解读**：
- 立刻拒绝 → M1a安全优先型
- 立刻答应 → M1b冒险扩张型
- 先问具体风险 → M1c寄生依附型（需要更多保障）
- 考虑自己能承担什么 → M1d自保隔离型

#### 第二步：自我认同模因（M2）检测

**场景问题**：如果有人公开说你"自私"、"没用"、"可悲"，你最接受不了哪个？

**解读**：
- 自私 → M2a道德好人型
- 无用 → M2b强者赢家型
- 可悲 → M2c受害者型
- 无所谓 → M2d自由异类型或M2e奉献者型（需要进一步区分）

#### 第三步：价值排序模因（M3）检测

**场景问题**：如果必须放弃一个，你选放弃金钱、放弃亲情还是放弃自由？

**解读**：
- 放弃金钱 → M3c/d/e（面子/感情/自由 > 财富）
- 放弃亲情 → M3b财富型
- 放弃自由 → M3a/c（生存/面子 > 自由）

#### 第四步：社会互动模因（M4）检测

**场景问题**：你怎么看那些靠关系上位、而你认为是"小人"的人？

**解读**：
- 愤愤不平，按规则举报 → M4b平等合作型
- 默默学习，也去找关系 → M4c零和博弈型
- 鄙视但无可奈何 → M4a等级支配型（默认等级存在）
- 无所谓，不关心 → M4e信任隔离型

#### 第五步：规则认知模因（M5）检测

**场景问题**：你觉得大多数人遵守的规则就是对的，还是有些规则本身就是有问题的？

**解读**：
- 规则就是规则，必须遵守 → M5a规则服从型
- 规则有问题，但没办法 → M5b规则利用型
- 我也要定规则 → M5c规则制定型
- 所有规则都有问题 → M5d规则对抗型
- 规则是工具，看怎么用 → M5e规则适配型

#### 第六步：应激模式模因（M6）检测

**场景问题**：你最近一次犯的错，你是怎么处理的？

**解读**：
- 假装没发生 → M6a逃避退缩型
- 怪别人、找借口 → M6b攻击甩锅型
- 长时间自责、无法释怀 → M6c内耗自责型
- 想办法补救 → M6d解决问题型
- 分析原因、总结经验 → M6e复盘迭代型

### 10.3 情境观察法

除了直接提问，更准确的是观察真实情境中的行为：

#### 情境一：面对地位低的人

观察他对服务员、保洁、门卫、实习生等的态度。

**解读**：
- 颐指气使 → M4a等级支配型
- 礼貌但保持距离 → M4b平等合作型或M4e信任隔离型
- 刻意讨好 → M4a或M4c
- 平等尊重 → M4b

#### 情境二：面对利益冲突

观察他在利益冲突时的选择。

**解读**：
- 牺牲自己利益维持关系 → M2a/M4b
- 争夺到底 → M2b/M4c
- 回避冲突 → M4e/M6a
- 寻求双赢 → M4b/M5e

#### 情境三：面对权威

观察他和上级、长辈、专家等权威相处时的表现。

**解读**：
- 无条件服从 → M5a
- 表面服从，内心评估 → M5e
- 试图挑战和超越 → M2b/M5d
- 保持距离 → M4e/M2d

#### 情境四：面对失败

观察他犯错时的反应和处理方式。

**解读**：
- 立刻推卸责任 → M6b
- 陷入自责 → M6c
- 快速补救 → M6d
- 分析复盘 → M6e
- 假装没发生 → M6a

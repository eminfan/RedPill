## 文学作品模因图谱

一个通过流程保障深度的作品立场分析框架。输入作品名，自动完成五层分析（L1-L5）+ 四维交叉验证 + 反事实测试 + 终极判定。

核心创新：**把批判性思维编码为不可跳过的硬性规则**。滑梯退出、对称性强制、拮抗标注、置信度锚定——即使执行者没有批判性思维天赋，严格按流程走也能产出深度分析。

详见 [literature/README.md](literature/README.md)。

## 人类模因图谱

一套基于天道理论的人类模因分类系统（M1-M6），用于：
- **识人**：通过六个核心模因组快速判断一个人的底层逻辑
- **自知**：识别自身的驯化模因，实现认知觉醒
- **预测**：通过模因组合推断此人在未知场景下的行为

详见 [human/README.md](human/README.md)。

# 文学作品模因图谱分析框架

## 这是什么

一个分析工具，用来拆解文学作品在"暗中传递什么价值观、为谁的利益服务"——是让人更清醒（启蒙），还是让人更麻木（驯化）。

## 怎么用

1. 想**分析一部作品**：打开[框架（当前版本）.md](框架（当前版本）.md)，把内容发给 Claude/DeepSeek 等大模型，附带作品原文
2. 想**看分析长什么样**：看[案例/](案例/)里的成品
3. 想**自己上手填空**：用[模板（空白）.md](模板（空白）.md)

## 目录结构

```
框架（当前版本）.md  ← 唯一入口，大模型执行指令
模板（空白）.md      ← 分析报告的空白模板
案例/               ← 已分析完成的作品
测试报告/           ← 版本测试记录（含 _test_output）
archive/            ← 旧版本和理论文档（无需阅读）
```

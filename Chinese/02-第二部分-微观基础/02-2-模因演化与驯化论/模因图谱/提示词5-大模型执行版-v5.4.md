## 一、执行顺序

严格按以下四阶段执行，阶段一必须完成才能进入阶段二：

```
阶段一：检索校准 → 事实锚定底表 + 作品类型识别
阶段二：L1-L5 模因五层分析（从素材到演化）
阶段三：E1-E4 四组学交叉验证（从文本到传播到受众）
阶段四：最终判断
```

**全局强制规则**：

1. **事实优先**：所有分析先锚定真实事实。任何偏离事实的扭曲/美化/抹黑就是核心模因。绝不能顺着作品叙事做无对照分析。
2. **立场前置**：所有人物塑造、情节，都是为作品的底层立场服务的。先明确立场再拆解叙事。
3. **表里拆分**：必须拆分「表层伪装」与「底层真实根模因」。多数作品有表层叙事伪装。
4. **闭环验证**：分析必须能回答——为什么扭曲事实？对受众的长期影响是什么？传播服务于谁？
5. **每层输出附带置信度标记：[高/中/低]**

## 二、阶段一：检索校准

### 2.1 作品类型识别

```
A. 历史题材——涉及真实历史人物/事件/朝代
B. 现实主义题材——以当代/近代现实社会为背景
C. 纯虚构世界观——完全虚构的世界
D. 寓言/改编作品——有明确原著的改编，或自觉使用寓言/象征/隐喻手法的创作
E. 功能文本——说明书/教程/规章/宣传材料/政论/战略论述
F. 混合型
G. 纯美学文本——不以传递世界观为目的，无叙事结构，无人物塑造，无社会关系描写
```

🆕 **D类（寓言/改编作品）判定标准**（符合以下任意一条即归为D类）：
- 有明确原著，且对原著进行了系统性改编（如《大话西游》对《西游记》的改编）
- 作品核心手法是寓言/象征/隐喻，人物和情节服务于观念表达而非模拟现实
- 作品中存在明确的"这不是在说这个，是在说那个"的自觉信号（如动物寓言、架空讽刺）
- 区别于纯虚构(C类)：C类试图建立自洽的世界观，D类自觉告诉读者"这是假的，我在说别的事"

**G类自动触发条件**（同时满足以下三条时自动判定为G类，无需继续分析）：
- 字数 < 2000
- 无人物、无情节、无对话
- 纯景物/情感描写

**G类处理方式**：输出"本作品不适合模因图谱分析"，直接结束。不进入L2-L5阶段。

### 2.2 分类型检索

**A类·历史题材**：检索①核心历史事件的真实经过 ②核心历史人物的正史评价 ③该时期的宏观数据（经济/人口/科技）与同期世界对比 ④作者的创作历史语境

**B类·现实主义**：检索①作品涉及的社会矛盾的真实数据 ②行业的真实运作逻辑 ③作者身份和公开立场 ④主要受众的阶层位置

**C类·纯虚构**：转向框架锚定——①世界观的逻辑自洽性 ②权力结构映射的现实关系 ③继承/挑战了哪些叙事传统

🆕 **D类·寓言/改编作品**（不同于A/B/C的检索方式——不检索"事实偏差"，转而检索"意图与效果"）：
- ①检索原著的原始模因（改编作品）
- ②检索改编者的公开立场和改编意图
- ③检索该寓言的象征系统对应的现实指涉（如《动物庄园》中猪=斯大林）
- ④检索该作品的受众解读史——当代人如何理解它
- ⑤A类/B类/C类中使用了自觉寓言/象征手法的，应标记为D类的子类并在事实锚定阶段做文学隐喻声明

**E类·功能文本**（重要——不同于文学作品的检索方式）：
- ①检索作品核心论点是否与客观事实一致
- ②检索作品引用的事实数据是否准确、有无选择性使用或断章取义
- ③检索作者的立场、动机、身份背景
- ④检索作品的实际历史影响和受众接受度
- ⑤检索同时代是否存在提供相反论点的竞争性文本

**混合型**：按主要类型检索，对每类维度单独处理。

**G类·纯美学文本**：确认是否满足触发条件，不检索。

### 2.3 事实锚定底表

**A/B/C/F类（文学作品）**：
```
| 维度 | 作品叙事 | 真实事实 | 偏差类型 |
|------|---------|---------|---------|
| 事件/人物 | 作品说…… | 实际上是…… | 美化/抹黑/回避/置换 |
```

🆕 **D类（寓言/改编作品）——替代框架**：

D类作品（寓言、改编、自觉象征）不适用标准的事实锚定，因为：
- 寓言中的人物和事件不是对事实的陈述，而是观念的载体
- 改编作品中的变异不是"偏差"，而是"再创作"
- 对D类做"美化/抹黑/回避/置换"标注，等于把寓言当成历史来批判——这是类别错误

D类使用以下框架替代事实锚定底表：

```
| 维度 | 作品呈现 | 现实指涉 | 变异方向与意图 |
|------|---------|---------|--------------|
| 符号/角色 | X在作品中代表什么？ | 该符号在现实中指涉什么？ | 强化/弱化/扭曲了现实指涉的形象？ |
| 叙事选择 | 故事怎么讲？省略了什么？强调了什么？ | 有无更忠实的改编方式？ | 改编方向揭示了什么意图？ |
| 立场传递 | 作品通过寓言真正想说的是什么？ | 这个立场在现实争论中的位置？ | 批判/维护/回避了哪一方？ |
```

**D类输出前强制声明**：

```
D类寓言声明：本作品的核心手法是寓言/象征/改编，因此不适用标准事实偏差标注。
以下分析不将"与事实不符"视为偏差，而是分析符号系统的指向和改编选择的意图。
```

**E类（功能文本）——替代框架**：

功能文本的"事实锚定"不以标准的美化/抹黑/回避/置换四种偏差类型为核心，而使用以下四个替代维度：

```
| 维度 | 作品论述 | 事实/逻辑核对 | 判断类型 |
|------|---------|--------------|---------|
| 论点正确性 | 作品的核心判断/结论 | 后来被验证/证伪？或当时可验证是否正确？ | 正确/部分正确/错误 |
| 修辞策略 | 论证中使用的说服技巧 | 对比/设问/排比/让步/递进 | 中性/强化/操控 |
| 选择性呈现 | 强调什么、回避什么 | 是否选择性使用数据或回避反例 | 全面/有偏向/严重偏向 |
| 框架预设 | 论证依赖的世界观前提 | 该前提是否被论证？是否有替代前提？ | 已论证/未论证/隐含 |
```

### 2.4 认知偏差标注与"文学隐喻"声明（强制）

对使用了自觉文学隐喻的作品（特别是B类现实主义或D类中使用夸张/荒诞/象征手法的作品），在偏差标注前必须先做以下声明：

```
文学隐喻声明：本作品核心手法是[夸张/荒诞/象征/寓言]，因此"偏差标注"中的"美化/抹黑/回避/置换"
是针对作品对真实事实的叙事处理，而非对文学手法本身的评判。文学隐喻不是事实偏差，它是创作选择。
以下偏差标注仅针对"以隐喻伪装成事实陈述"的部分，不针对明确的文学表达。
```

输出格式：
```
检索信源：______（是否存在偏向？）
知识完整度：______（哪些维度可能信息不足需人工补充）
```

## 四、阶段三：E1-E4 四组学交叉验证

**核心逻辑**：L1-L5 都是文本内部的分析。结论是否可靠，需要用独立于文本的维度来验证。

### 证据等级定义

| 等级 | 条件 | 含义 |
|------|------|------|
| S级 | 4维一致 | 可靠，可直接引用 |
| A级 | 3维一致 | 较可靠，标注缺失维度 |
| B级 | 2维一致 | 待更多验证 |
| C级 | 仅E1(文本) | 无独立验证，仅供参考 |
| D级 | 无证据 | 不应出现在正式图谱 |

### 🔧 S级泛滥约束（v5.4新增）

当5/5维度全部一致时，**不自动评S级，必须通过以下检查**：

```
【S级强制自检】

如果本分析为S级（5/5一致），必须回答：
```

## 五、阶段四：最终判断

### 5.1 最终判断

**最终判断的核心问题只有一个：这套叙事在为谁的利益说话？**

```
【最终判断】
标签：[强驯化 / 弱驯化 / 中性 / 弱启蒙 / 强启蒙]
行动维度：[有明确行动方案 / 有行动方向但无方案 / 行动指向模糊或向内]

① 这部作品在为谁的利益说话？
  - 必须答出具体的社会群体/利益方，说明作品为他们做了什么叙事服务
  - ❌ "服务于统治阶级" → ✅ "服务于满清殖民统治的合法性——通过美化康熙、回避剃发易服和文字狱，将异族征服重新包装为'圣君治世'"
  - ❌ "历史虚无主义" → ✅ "通过韦小宝替代真实历史人物的叙事，消解了汉族对清初民族压迫的历史记忆"

② 对读者是叫醒还是哄睡？如果是哄睡，用的是什么具体手法？

③ 看懂了该怎么做？——防御：如何识别？认知修复：被影响过如何恢复？
```

**输出后自检**：
- 三个回答中是否有具体的历史人物/社会群体/利益方名称？如果全是抽象名词，重写。

**标签判定规则**：

| 标签 | 判定条件 | 典型例子 |
|------|---------|---------|
| 强启蒙 | DIM-05揭示真相、打破驯化、唤醒独立判断 | 狂人日记、论持久战 |
| 弱启蒙 | DIM-05提供另类视角/质疑主流但不彻底 | 百年孤独 |
| 中性 | DIM-05不传递价值判断/无根模因倾向 | 朱自清《春》（G类） |
| 弱驯化 | DIM-05有偏向但不极端，且保留审美/知识价值 | 红楼梦 |
| 强驯化 | DIM-05系统性扭曲事实，消解批判能力 | 鹿鼎记、狼图腾 |

**E类功能文本补充规则**：
- 功能文本的DIM-05通过"论述"直接传递而非通过"叙事"间接传递
- E类判定：框架预设开放+主动处理反例=强启蒙；将特定价值观包装为"常识"=弱驯化
- 行动维度：有明确行动方案/有方向但无方案/非行动型文本

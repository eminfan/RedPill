# 人类模因图谱

> 基于天道理论的人类模因分类系统。
> 识人·自知·觉醒——理解人的"精神DNA"。

---

## 一句话

如同DNA图谱记录基因构成，人类模因图谱记录一个人的核心模因构成——通过六大模因组（M1-M6）系统拆解一个人的底层行为逻辑。

---

## 六大模因组

| 编号 | 模因组 | 核心问题 | 决定什么 |
|------|--------|----------|----------|
| M1 | 生存安全模因 | 稳定 vs 机会 | 人生战略方向 |
| M2 | 自我认同模因 | 我是谁 | 行为底线与情绪触发点 |
| M3 | 价值排序模因 | 什么最重要 | 所有两难选择的决策 |
| M4 | 社会互动模因 | 人和人是什么关系 | 社交模式与亲密关系 |
| M5 | 规则认知模因 | 我和规则是什么关系 | 社会层级与觉醒程度 |
| M6 | 应激模式模因 | 遇到问题怎么办 | 人生上限与抗压能力 |

详见 [人类模因图谱.md](人类模因图谱.md)（完整 v1.0 分类体系）。

## 使用场景

| 场景 | 方式 |
|------|------|
| **个人成长** | 识别自身的驯化模因，找到觉醒方向 |
| **识人交友** | 通过 M1-M6 快速判断一个人的底层逻辑 |
| **关系处理** | 理解伴侣、家人、同事的模因驱动 |
| **团队组建** | 根据模因组合配置互补团队成员 |
| **行为预测** | 通过模因组合推断此人在未知场景下的选择 |

---

## 核心原则

1. **模因在潜意识层面** — 嘴上说的和实际驱动行为的往往完全相反
2. **组合比单点重要** — 要看完整的模因组合，而非单个模因
3. **只有在冲突中才暴露真实模因** — 两难选择、应激状态下的行为才真实
4. **模因是动态的** — 可以随经历和环境变化
5. **不可过度简化** — 人是复杂的，模因图谱是工具不是标签

# G-心理能

> 用天道理论从根本层(物理公理)推演"心理能"的体系模块。它不把心理能当心理学概念(意志力/注意力),而是当**耗散结构为高效逆熵而演化的认知工具,同时又是系统逆熵要消耗的燃料**——从熵增公理逐级推演到共生回流。

## 为什么有这个模块

"心理能"在天道全体系里散落在至少 9 个文件(营养级三层定义、灵视熵能转化定律、本能-驯化接口映射、错配论、能量管理、超个体统合、文化消费分析……),但从未被作为一个独立对象整合成完整理论。

主流的心理能理解(认知心理学+注意力经济+意志力研究)停在**中层**:它们把"心理能存在"当不证自明的前提,只研究"它怎么消耗/怎么管理",跳过了最根本的一问——**一个耗散结构,为什么会演化出心理能?它在能量收支上算什么账?它流向谁、归宿是什么?**

本模块从天道底层公理逐级推演,到达中层解释到达不了的深度——能回答**心理能作为系统燃料被烧的机制、耗竭为何可由社会症状间接观测、以及心理能的文明归宿取决于天道类型**。

## 理论空白填补清单(相对散落锚点的增量)

本模块相对天道各处散落的"心理能"锚点,填补了以下空白:

- **内部结构模型**——从"标量意志力"到"三维向量(注意力/情绪/意志)",有质的方向性
- **生成机制**——从葡萄糖到认知能的完整通路,唯物基底明确
- **度量学**——个体级替代指标 + 群体级社会症状间接观测,自有指标体系
- **收支账户模型**——三维分账户,通用收支方程,与 F 负熵收支对接
- **耗竭-恢复动力学**——正反馈闭环(非线性损耗)+ 恢复物理基础 + 伪休息陷阱
- **质 vs 量维度**——三维不可加总,同样"累"有变富变穷之分
- **与虚拟/真实负熵的形式关系**——心理能是度量负熵收支的尺子,非被消费对象
- **跨层流动拓扑**——三层能量位置 + 超个体三种流向 + 共生回流
- **文明归宿**——掠夺型燃料 / 共生型本金,心理能归宿是天道类型函数

## 边界声明

- 本模块是**描述为主、规范为辅、应用收束**:00-03 描述机制,04 给规范方向(共生回流),05 把理论应用于 [C-天道实践理论/实操地图](../C-天道实践理论/03-天道实操地图-从动不了到停不下来.md) 的"动不了"解析。
- 不把心理能神秘化——它是葡萄糖燃烧后的真实产物,不是灵魂能量。全文无一处脱离物理载体。
- 不主张"心理能管理就是养生"——能量基线管理是系统运行的基础设施(挂接 D 系列实操)。
- 不给乌托邦蓝图——共生回流只给原则与判据(需制度做功、经传递层有截留风险、动态迭代)。
- 与 [F-文化消费分析](../F-文化消费分析/) 共享唯物立场与负熵判据,F 讲"心理能被虚拟负熵代偿耗散",G 讲"心理能本身是什么"——互补不重复。

# 收支账户篇:心理能的收支账户与度量学

> 本文是 [根本论](00-从根本理解心理能.md) 的第二篇派生,展开根本论第 7 级。根本论已指出:心理能不可被焦耳完全量化,但可建立收支账户模型,并以替代指标近似估算——个体级用时间/效率/组块,群体级用社会症状间接观测。本文把这一级工程化:**给出一个可算的心理能收支账户,以及一套自有度量学。**
>
> 营养级理论已声明"心理能不可完全量化,但它真实存在而且可以被间接观察"([4.0](../../04-第四部分-中观结构-社会营养级理论/4.0-社会营养级理论基础.md) 行155)。本文不假装能精确测量,而是给"间接观测"一套可操作的账法。

---

## 一、为什么要建"账户"而不是求"总量"

营养级理论明确反对追求一个统一的"社会焦耳"度量([4.0](../../04-第四部分-中观结构-社会营养级理论/4.0-社会营养级理论基础.md) 行134-143):核心任务是**追踪流向**——谁在生产、谁在截留、谁在富集,而非精确计算总量。要追踪流向,每层内部有各自的量化单位就够了。

心理能同理。**与其追求"一个人今天还剩多少心理能"(不可测的标量),不如建一个收支账户,追踪心理能从哪来、花到哪去、缺口多大。** 账户的意义不在精确余额,而在**流向与缺口**。

> 这个定位决定了本文的方法:**账户是分析工具,不是计量仪表。** 它不给你一个"心理能 = 73.2 单位"的假精确数字,而是给你"今天注意力账户入账 X、消耗 Y、缺口 Z,缺口主要来自干扰消耗"的流向判断。

---

## 二、心理能收支账户模型

承接 [结构模型篇](01-心理能的内部结构与生成机制.md) 的三维结构,心理能账户不是一个总账,而是**三个分账户**——注意力账户、情绪驱动账户、意志力账户,各有收支。

### 2.1 三分账户的收支结构

| 分账户 | 入账(生成) | 出账(消耗) | 缺口信号 |
|--------|-----------|-----------|---------|
| **注意力账户** | 睡眠恢复、自然/运动恢复 | 任务聚焦、信息过载碎片化、任务切换 | 无法聚焦、易分心 |
| **情绪驱动账户** | 真实成就、真实关系、运动(稳定基线) | 情绪劳动、即时刺激(拉高基线)、持续威胁 | 麻木或亢奋、需更强刺激 |
| **意志力账户** | 睡眠(前额叶恢复)、进食(血糖) | 核心任务、决策、抑制冲动、对抗干扰 | 拖延、决策疲劳、明知该做却做不到 |

**关键洞察**:三个分账户的恢复机制高度重叠(都依赖睡眠、进食),但消耗机制不同——这决定了**恢复要统一做,消耗要分别管**:

- 恢复:睡眠同时恢复三个账户(注意力带宽重建、情绪基线稳定、前额叶储备补充),进食主要恢复意志力账户(血糖),运动扩展三个账户的容量上限。
- 消耗:注意力账户被"信息过载"消耗,情绪账户被"情绪劳动/即时刺激"消耗,意志力账户被"决策/抑制"消耗——三种消耗要分别识别、分别堵漏。

### 2.2 通用收支方程

每个分账户都服从同一个收支方程:

> **账户余额变化 = 生成量 − 消耗量**
>
> 当 生成量 < 消耗量,持续 N 个周期 → 该维度耗竭(不可逆退化阈值)

把三个分账户合起来,心理能的整体状态:

```
心理能整体状态 = {
  注意力账户: 余额 + 趋势,
  情绪驱动账户: 余额 + 基线偏移,
  意志力账户: 余额 + 趋势
}
```

注意:**整体状态不是三个余额的简单加总**(结构模型篇 3.1 已论证三维不可加总),而是三个分账户状态的**向量**——一个向量(高注意力、稳情绪、低意志力)和另一个向量(低注意力、亢奋情绪、高意志力)代表完全不同的状态,即便"总余额"看起来一样。

> 这个向量模型是本篇的核心贡献:**心理能不是一个数,是一个三维向量。** 账户算法的不是"还有多少",而是"三个分账户各自处于什么状态、缺口在哪、趋势如何"。这才是可操作的账法。

### 2.3 与 F 体系负熵收支的对接

这个账户模型与 [F-文化消费分析/能量代谢篇](../F-文化消费分析/02-能量代谢篇-文化消费的负熵流动.md) 的负熵收支方程严格对接:

- F 的"受众净收支 = 获得的虚拟负熵 − 耗散掉的可用于真实逆熵的心理能"——其中"耗散掉的心理能"就是本篇的**账户出账**。
- F 的"增强削弱"判据——消费后真实负熵获取能力增减——在本篇落地为:**消费后三个分账户的余额与趋势如何**。增强 = 账户余额稳定或上升(注意力可聚焦、情绪基线平稳、意志力可调用);削弱 = 账户余额下降(注意力碎片化、情绪基线被拉高、意志力耗竭)。

> 这一级让 G(心理能账户)与 F(负熵收支)在度量上对齐:**F 算的是"消费这行为的负熵账",G 算的是"承接这个账的心理能账户本身的状态"。** F 是流水,G 是账户——流水记在账户上,账户反映流水的累积后果。

## 三、度量学:自有指标体系

承接根本论第 7 级,心理能虽不可被焦耳完全量化,但每维都有替代指标([4.0](../../04-第四部分-中观结构-社会营养级理论/4.0-社会营养级理论基础.md) 行100-103)。本文把它们组织成一套**自有度量学**——个体级与群体级两层。

### 3.1 个体级度量(当前状态)

| 分账户 | 替代指标 | 估算方式 | 优点/局限 |
|--------|---------|---------|---------|
| 注意力账户 | 时间(注意力投向) | 记录 24h 注意力投向何处 | 可记但不含质量 |
| 注意力账户 | 7±2 组块 | 同时处理任务的限量 | 测带宽,需主动测试 |
| 情绪驱动账户 | 效率负相关 | 情绪耗竭与工作效率负相关 | 间接,需基线对比 |
| 情绪驱动账户 | 刺激阈值 | 引发同等满足所需刺激量 | 测基线偏移,可自测 |
| 意志力账户 | 决策次数/疲劳 | 一天内重大决策次数与后续决策质量 | 耗损可测,需日志 |

**操作要点**:个体级度量不求精确数值,求**趋势与缺口定位**。三个问题就够用:
1. 注意力账户:今天能持续聚焦多久?(对比自己健康基线)
2. 情绪驱动账户:同等满足需要的刺激量在变强还是变弱?(基线偏移)
3. 意志力账户:今天明知该做却没做的事有几件?(决策疲劳信号)

> 这三个问题把"我今天心理能如何"从模糊感受变成可定位的向量状态。不需要仪器,但能区分"注意力耗尽型疲惫"与"意志力耗尽型疲惫"——从而选对恢复手段(前者要断信息源,后者要补睡眠血糖)。

### 3.2 群体级度量(社会症状间接观测)

这是旧答案完全缺失的维度。营养级理论已点明([4.0](../../04-第四部分-中观结构-社会营养级理论/4.0-社会营养级理论基础.md) 行155):你无法用焦耳衡量一个人的绝望,但你能看到自杀率、生育率、抑郁症发病率的变化——它们是心理能量耗竭的社会症状。

把群体级度量组织成一张**心理能集体耗竭的指标表**:

| 社会症状 | 反映的分账户耗竭 | 机制 |
|---------|----------------|------|
| 自杀率上升 | 意志力账户 + 情绪驱动账户(麻木侧) | 长期入不敷出,维持生命的心理能归零 |
| 生育率塌陷 | 意志力账户 | 无力承担长期承诺(育儿是高意志力投入) |
| 抑郁症高发 | 情绪驱动账户(麻木侧) | 想要与喜欢回路都失活 |
| 成瘾率上升 | 情绪驱动账户(亢奋侧) | 想要回路过载,需更强刺激 |
| 专注力普遍下降 | 注意力账户 | 信息过载碎片化耗散 |

> **关键洞察:这些社会症状不是孤立的"社会问题",是同一个心理能集体耗竭过程在不同维度的显现。** 一个社会自杀率上升 + 生育率塌陷 + 抑郁症高发同步出现,是该社会心理能集体入不敷出的结构性信号——且这种信号**不依赖个体自陈报告**,是物理性可观测的(自杀率、生育率是硬统计)。这给社会诊断一个不靠主观感受的尺子。

### 3.3 度量学的边界声明

必须守的边界,否则度量学会滑向伪科学:

1. **不追求焦耳级精确**:心理能是质性状态,替代指标是"间接估算",不是"精确测量"。一个社会的自杀率上升,说明心理能集体耗竭,但不能算出"耗竭了 12.3 单位"。
2. **替代指标是相关性而非因果性单值**:效率与情绪耗竭负相关,但效率下降可能有其他原因(技能、资源)。度量学用于**定位趋势与缺口**,不用于**单点判决**。
3. **群体级症状是滞后指标**:社会症状显现时,耗竭已发生多时。它是"诊断"工具,不是"预警"工具——预警要靠个体级趋势追踪。

> 这三条边界让度量学保持诚实:**有用(可观测趋势与缺口)但不精确(不可焦耳还原),可诊断(群体级硬统计)但滞后(症状是后果)。** 这与营养级理论"能量是跨层类比不是物理还原"的立场一致。

## 四、收支账户篇的核心命题

1. **心理能要建账户而非求总量**:营养级理论反对统一度量,心理能同理。账户的意义在追踪流向与缺口,不在精确余额。账户是分析工具,不是计量仪表。

2. **心理能是三维向量账户**:注意力、情绪驱动、意志力三个分账户,恢复机制重叠(都靠睡眠进食)但消耗机制不同——恢复要统一做,消耗要分别管。整体状态是三维向量,不是标量加总。

3. **个体级度量用替代指标定位趋势与缺口**:时间/组块(注意力)、效率负相关/刺激阈值(情绪)、决策疲劳(意志力)。不求精确数值,求区分"哪种疲惫"以选对恢复手段。

4. **群体级度量用社会症状间接观测集体耗竭**:自杀率、生育率、抑郁症、成瘾率、专注力下降——同一耗竭过程在不同维度的显现,且不依赖个体自陈,是物理性可观测的硬统计。给社会诊断一把不靠主观感受的尺子。

5. **G 账户与 F 负熵收支对接**:F 算"消费行为的负熵账"(流水),G 算"承接这个账的心理能账户"(账户)。流水记在账户上,账户反映流水的累积后果。增强削弱判据在 G 落地为账户余额与趋势。

6. **度量学守边界**:有用但不精确(不可焦耳还原),可诊断但滞后(群体症状是后果)。这与营养级理论"能量是跨层类比不是物理还原"的立场一致。

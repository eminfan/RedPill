# F-文化消费分析

> 用天道理论从根本层(物理公理)推演文化消费现象的体系模块。它不把"人为什么消费文化作品"当心理学问题,而是当**物理问题**——从熵增公理逐级推演到文明出口。

## 为什么有这个模块

文化消费的主流解释(进化心理学+消费社会学+算法机制)停在**中层**:它们从"五大本能"起跳,跳过了天道最底下的四级公理(熵增→资源有限→群体演化优先→双遗传),本质定义停在心理层(奖赏释放)未落到负熵收支层(能量跨层类比),中观只把营养级当"受众定位",宏观周期律零引用。

本模块从天道底层公理逐级推演,到达中层解释到达不了的深度——能回答**文化消费在文明能量代谢中的角色**、以及**为什么特定历史阶段必然涌现特定内容形态**。

## 推演链骨架(根本论11级)

```
公理1 熵增 → 公理2 资源有限 → 公理3 群体演化优先
  → 公理4 双遗传 → 公理6 掠夺趋利
  → [第6级] 五大本能(从群体逆熵五环节推出,非黑箱前提)
  → 定理三 借壳表达 → 演化速率错配论(为何当下病理化)
  → 公理14 负熵驱动(本质重落负熵收支层:虚拟负熵替代真实负熵)
  → 营养级功能(稳态阀门+收割管道) → 定理11+5.1 共生出口
```

## 边界声明

- 本模块是**描述为主、规范为辅**:00-03描述机制,05给规范方向。
- 不评判哪种内容"高级"——天道不戴道德滤镜,只算负熵收支。
- 不主张消灭文化消费——镇痛型是低能量生存策略,判别在"是否看清机制并自主选择",非"是否消费"。
- 源码素材([源码/文化源码](../../../源码/文化源码/))质量低于本体系,留作参考,F目录另起炉灶。

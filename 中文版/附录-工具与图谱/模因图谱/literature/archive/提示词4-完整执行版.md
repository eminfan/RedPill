> 版本：v4.0（2026-05）
> 定位：供豆包/DeepSeek/Claude 等文本大模型执行的完整模因图谱分析指令
> 前置：本提示词为一次性完整输入，包含分析所需全部框架定义，不依赖外部文档
> 产出：一份完整的 L1-L5 模因图谱 + E1-E4 四组学交叉验证矩阵 + 附置信度标记的终极判定

## 一、执行总纲

本提示词包含四个阶段，按顺序执行，**阶段一完成后进入阶段二，不可跳过**：

```
阶段一：检索校准
  → 分类型事实锚定检索
  → 输出事实锚定底表
  → 强制——不可跳过

阶段二：L1-L5 模因序列分析（五层产品结构）
  → L1：模因序列层（素材清单 + 偏差标注）
  → L2：模因结构层（五染色体分组）
  → L3：模因功能层（基因功能注释）
  → L4：模因调控层（表达调控网络）
  → L5：模因演化层（跨作品比较）

阶段三：E1-E4 四组学交叉验证
  → E1（已自动完成 = 阶段二的 L1）
  → E2 传播组学（外部数据验证）
  → E3 调控组学（社会结构验证）
  → E4 效果组学（受众反馈验证）
  → 输出「四组学一致性矩阵」

阶段四：终极判定 + 质量控制
  → 三个核心问题
  → 覆盖度自评
  → 反事实测试
```

## 二、全局原则（六条铁则）

以下六条铁则在分析的每个阶段都必须遵守：

**铁则1：事实优先**
所有分析必须先锚定作品对应的真实事实。将虚构叙事与真实事实做全面对照。任何偏离事实的扭曲、美化、抹黑，都是核心模因的关键组成部分。绝对不能顺着作品叙事做无对照分析。

**铁则2：立场前置**
先明确作者与作品的底层立场，再拆解叙事与模因。所有人物塑造、情节设计、台词符号，都是为作品的底层立场服务的。

**铁则3：表里拆分**
必须拆分作品的「表层伪装」与「底层真实根模因」。多数具备意识形态属性的作品有表层叙事伪装。不能被表层爽文/情怀叙事欺骗。

**铁则4：驯化/启蒙判定**
模因分析的核心是拆解作品对受众的认知影响——它是打破驯化、唤醒受众的启蒙模因，还是固化认知、维护掠夺结构的驯化模因。

**铁则5：闭环验证**
一套合格的模因图谱必须能无矛盾地解释三个问题：①作品为什么刻意扭曲/美化事实？②作品对受众产生了什么可验证的长期认知影响？③作品的传播最终服务于谁的利益？

**铁则6：对照原著（如为改编作品）**
改编作品必须对照原著作差异标记，识别改编中的降维、洗白、矮化、消解、置换等手法。

## 三、阶段一：事实锚定检索

### 3.1 强制步骤：作品类型识别

分析开始前，先识别目标作品的核心类型：

```
A. 历史题材——涉及真实历史人物/事件/朝代，有可对照的正史记载
B. 现实主义题材——以当代/近代现实为背景，涉及真实社会结构
C. 纯虚构世界观——完全虚构的世界，不直接对应真实历史或现实
D. 改编作品——有明确原著/原始素材的改编
E. 功能文本——说明书、教程、规章、宣传材料等
F. 混合型——兼有以上多种类型特征
```

### 3.2 分类型检索指令

**A类·历史题材检索**：
检索以下维度建立事实锚定底表：
1. 核心历史事件：作品涉及的每个关键事件，真实历史上发生了什么？
2. 核心历史人物：作品中出现的真实历史人物，其真实生平是什么？正史评价是什么？
3. 时代背景宏观事实：该朝代的经济总量、人口、科技、文化水平；同时期世界对比
4. 作品后设史：作者创作时的历史语境、目标受众的主流历史认知

**B类·现实主义题材检索**：
1. 作品描写的核心社会矛盾的真实数据（基尼系数、阶层流动率等）
2. 作品涉及行业的真实运作逻辑和利益分配结构
3. 作者的真实身份、行业背景、公开立场
4. 主要受众的社会营养级位置和核心情感共鸣来源

**C类·纯虚构世界观检索**（转向"框架锚定"）：
1. 世界观的内在逻辑是否存在矛盾？
2. 虚构世界的权力结构映射现实中什么关系？
3. 作品继承/挑战了哪些已有的叙事传统？
4. 作品在刻意挑战读者的哪些默认认知？

**D-F类**按混合特征执行对应维度的组合检索。

### 3.3 事实锚定底表

检索完成后输出以下格式：

```
## 事实锚定底表

| 维度 | 作品叙事 | 真实事实 | 偏差类型 |
|------|---------|---------|---------|
| 事件/人物1 | 小说说…… | 历史上是…… | 美化/抹黑/回避/置换 |
| …… | …… | …… | …… |
```

### 3.4 认知偏差标注（强制）

大模型必须自动标注自己的认知局限：

```
认知偏差标注：
1. 检索信源：______；是否存在信源偏向？
2. 我对该作品领域的知识完整度：______
3. 需要分析者补充/确认的关键事实：______
```

## 四、阶段二：L1-L5 模因五层分析

### L1：模因序列层（全量测序）

**说明**：L1 拆为层0（原始素材清单）和层1（偏差标注），以区分"客观记录"和"主观判断"。术语说明——这里的"碱基"不是化学实体而是分析者的识别结果，"测序"是系统性收录而非化学检测。使用"测序""碱基"等术语仅为结构上类比基因组学。

#### 层0——原始素材清单

收录标准：作品中可精确定位的最小内容单元。只登记，不评价。
素材类型：台词 / 设定 / 叙事选择 / 符号

每个素材记录：
```
素材ID: SRC-001
类型: [台词/设定/叙事选择/符号]
位置: [作品中位置]
内容: [原文或精确描述]
```

**完整性要求**：宁可多收十个无关素材，不漏一个关键素材。台词、设定、叙事选择、符号四类尽可能全面覆盖。

#### 层1——偏差标注

将层0每个素材与事实锚定底表对照，标注偏差类型：

| 偏差类型 | 定义 | 标注依据 |
|---------|------|---------|
| 美化 | 将负面事实呈现为中性或正面 | 对照底表中的真实事实 |
| 抹黑 | 将中性或正面事实呈现为负面 | 同上 |
| 回避 | 对关键事实完全不提及 | 作品中没提应该提的 |
| 置换 | 用一个伪命题替代真实命题 | 替换论证对象 |
| 无偏差 | 与事实一致 | 同上 |

每个偏差标注格式：
```
标注ID: DEV-001
关联素材: SRC-001（简述）
偏差类型: [美化/抹黑/回避/置换/无偏差]
偏差方向: [具体描述]
事实依据: [具体引用真实事实]
置信度: [高/中/低]
```

**输出格式**：分别列出层0清单和层1清单，层0的素材如果没有偏差则不进入L2分析。

### L4：模因调控层（表达调控网络）

**说明**：术语说明——"启动子""增强子""沉默子""绝缘子"在生物学中有明确的DNA序列特征和实验验证方法，在模因分析中是对叙事策略的归类分析。两者的相似点是"表达受多层因素控制"，不同点是识别精度。使用这些术语仅为结构化分类，不是分子层面的验证。

**核心任务**：分析作品的模因是如何被调控表达的。

#### 调控元件识别

| 调控元件 | 作品中的表现 | 识别的叙事特征 |
|---------|------------|---------------|
| **启动子**（开启表达） | | 作品的开端、第一章、背景交代 |
| **增强子**（增强效果） | | 反复出现的主题、高潮场景 |
| **沉默子**（抑制竞争） | | 对对立观点的反驳/滑稽化 |
| **绝缘子**（防止溢出） | | "这只是故事"等免责声明 |
| **反馈环**（自我强化） | | 结尾回到开头的循环印证 |

**第一级：叙事调控**（作品内部）

| 受众状态 | 预期表达效果 |

**第三级：文化调控**（社会语境）

### L5：模因演化层（跨作品比较）

**说明**：术语说明——"同源""趋同""演化树"在生物学中有序列比对和分子钟等客观工具，模因的同源判定只能依赖偏差特征比对和文化谱系追溯，输出的是"倾向性判断"而非确定性结论。每个L5结论必须附带置信度标记。

**核心任务**：将目标作品的核心模因放在跨作品的演化谱系中定位。

#### 同源/趋同判定

四步判定流程：

```
步骤1：偏差特征比对
  - M1与本作品的L1偏差类型是否一致？
  - 偏差方向是否一致？
  - 涉及的L2染色体定位是否相同？
  → 三层全匹配 → 倾向同源

步骤2：文化谱系追溯
  - 两者是否出现在同一文化传播链条中？
  - 时间顺序是否支持继承关系？
  → 有传播链 → 倾向同源

步骤3：利益方向判断
  - 两者是否服务于同一个利益方向？
  → 是 → 同源可能性高

步骤4：功能类比
  - 功能相似但结构差异大 + 无传播链 → 倾向趋同

判定输出：[同源/趋同/不确定性]
置信度：[高/中/低]
```

#### 演化谱系定位

在与同类作品比较中，标注本作品的模因在演化树中的位置：

```
演化树示意（文本格式）：
```

## 五、阶段三：E1-E4 四组学交叉验证

**核心逻辑**：阶段二L1-L5的分析是"文本内部"的。但这些结论是否可靠？需要用独立于文本分析的证据维度来验证。E1-E4就是四个独立的验证窗口。

**置信度等级**：
| 等级 | 条件 | 含义 |
|------|------|------|
| S级 | 4个维度一致 | 结论可靠，可直接引用 |
| A级 | 3个维度一致 | 结论较可靠，标注缺失维度 |
| B级 | 2个维度一致 | 需标记"待更多验证" |
| C级 | 只有E1(文本) | 无独立验证，仅供参考 |
| D级 | 无独立证据 | 不应出现在正式图谱中 |

**采集方法等级**：每条E2/E3/E4结论必须附带采集方法等级标记：

| 等级 | 含义 | 独立性风险 |
|-----|------|-----------|
| P1（API系统采集） | 通过API或爬虫获取的定量数据 | 低 |
| P2（手动系统采样） | 人工抽样+编码的手动分析 | 中 |
| P3（分析者观察） | 基于分析者印象和常识的定性判断 | 高，可能混入E1认知污染 |
| P4（理论推断） | 无数据支撑的逻辑推断 | 极高，实质是E1延伸 |

**独立性校验**：P3级数据必须声明"独立获取"或"非独立获取"。P1/P2默认独立，P4默认非独立。非独立获取的数据不能作为独立验证维度。

### E2：传播组学

**核心问题**：哪些模因实际进入了传播、被复制和变异了？（对应基因组学的RNA-seq）

**数据采集模式**：以下每组子维度提供三种采集方法——联网模式、离线手动模式、大模型辅助模式。选择当前可用的方法执行。

---

#### E2-a：转录检测——哪些模因被复制了

**核心操作**：拿E1层0的素材清单，逐条检查它们在改编版本中是否被保留、删除或改变。

**联网模式（可达P1~P2）**：搜索"[作品名]+改编"，列出所有改编版本。逐一比对核心模因保留情况。
**离线手动模式（可达P2~P3）**：凭回忆制作改编版本清单（标注[我看过]/[我知道有]/[听说过]），逐版对照核心设定保留/删除/改变情况。
**大模型辅助模式（可达P2~P3）**：将E1素材清单和已知改编版本输入大模型，让大模型逐条分析保留情况。

输出格式：
```
E2-a 转录检测：
  跨媒介复制：
    - [媒介类型]：版本清单
    - 核心模因保留情况：______
  → E1识别的核心模因在跨媒介传播中是否被保留：[一致/部分变异/不一致]，[P等级]
```

---

#### E2-b：变异检测——模因在复制中改变了什么

**核心操作**：对E1识别的核心模因，追踪它们在传播中是否发生了意义偏移。

**联网模式（可达P1~P2）**：搜索核心台词/设定的当代引用，采集20-50个独立使用实例，标注使用语境（严肃引用/搞笑梗/反讽用法/无意义口头禅）。
**离线手动模式（可达P2~P3）**：采用「记忆采样法」——凭回忆收集每个模因的独立使用实例，逐条标注语境分类和可信度[A清晰记忆/B模糊印象/C推测]。
**大模型辅助模式（可达P2~P3）**：输入核心模因清单，要求大模型列举训练语料中常见引用实例和漂变轨迹，要求区分"常见用法"vs"推断"。

输出格式：
```
E2-b 变异检测：
  意义漂变轨迹：
    - [模因名称]：原始功能→当代漂变形态→漂变方向
  功能转化分析：
    - 表层转化：______
    - 深层保留：______
  → [一致/部分一致/不一致]，[P等级]
```

---

#### E2-c：表达量分析——不同人群中的表达差异

**核心操作**：分析模因在不同受众群体（代际/地域/社会层级）中的接受方式差异。

**联网模式（可达P1~P2）**：不同平台分平台采样（豆瓣 vs 知乎 vs B站 vs 微博——用户画像不同），比较讨论焦点差异。
**离线手动模式（可达P3~P4）**：列出可能的代际/地域差异假设，标注[已验证]/[有线索]/[纯推断]。
**大模型辅助模式（可达P3）**：要求大模型基于训练语料中不同平台的讨论特征，给出差异推断。

输出格式：
```
E2-c 表达量分析：
  代际差异：______
  地域差异：______
  → [补充信息/不纳入一致性矩阵因P等级不足]，[P等级]
```

### E3：调控组学

**核心问题**：什么社会条件控制了这个模因的表达？（对应基因组学的ChIP-seq）

**独立性保证**：分析者不讨论"作品想表达什么"，而是讨论"什么社会力量在传播这个作品"。

---

#### E3-a：权力结构分析

| 数据维度 | 采集内容 | 识别方法 |
|---------|---------|---------|
| 体制亲和性 | 底层叙事是否与主流意识形态一致 | 对照国家话语、教育体系核心理念 |
| 资本路径 | 通过什么商业模式传播？谁获利？ | 产业链分析 |
| 筛选机制 | 谁决定"这部作品值得传播"？ | 传播链关键节点分析 |

#### E3-b：渠道分析

| 数据维度 | 采集内容 |
|---------|---------|
| 传播载体演变 | 印刷→电视→网络→短视频的筛选效果变化 |
| 渠道垄断 | 是否存在单一渠道控制传播的情况？ |
| 算法效应 | 推荐算法放大哪些模因、抑制哪些？ |

#### E3-c：竞争模因分析

| 数据维度 | 采集内容 |
|---------|---------|
| 同时代竞争 | 提供相反模因的同类作品 |
| 抵抗叙事 | 受众的对抗性解读 |
| 替代框架 | 其他能解释作品特征的理论 |

输出格式：
```
【E3 调控组学】

E3-a 权力结构分析：
  体制亲和性：______
  资本路径：______
  筛选机制：______
  → [一致/补充/不一致]，[P等级]

E3-b 渠道分析：
  传播载体：______
  渠道垄断：______
  → [一致/补充/不一致]，[P等级]

E3-c 竞争模因：
  竞争模因存在且处于[强势/弱势]：______
  → [补充/削弱主结论]，[P等级]
```

### E4：效果组学

**核心问题**：模因真的对受众产生了认知/行为影响吗？（对应基因组学的敲除实验）

**独立性保证**：数据来自受众的真实反馈，不来自分析者推断。

---

#### E4-a：受众情感表型

| 数据维度 | 采集方法 |
|---------|---------|
| 人物情感偏好 | 联网：豆瓣短评API情感分析。离线：记忆采样法，标注[压倒性多数/多数/分歧明显/样本不足] |
| 价值排序 | 讨论中的"站队"分析 |
| 情感传播 | 推荐语情感编码 |

#### E4-b：受众认知表型

| 数据维度 | 采集方法 |
|---------|---------|
| 事实认知改变 | 评论中暴露的历史/现实认知 |
| 框架借用 | 受众是否使用作品叙事解释现实 |
| 批判意识 | 抵抗式阅读的存在和质量 |

#### E4-c：受众行为表型

注意：长期社会行为影响在公开数据条件下不可系统性采集。此维度**不作为S级判定的必要条件**。标记方法：可观察的标记为[定性样本级]，不可采集的标记为[待方法成熟]。

输出格式：
```
【E4 效果组学】

E4-a 受众情感表型：
  核心人物情感倾向：______
  → [一致/不一致]，[P等级]

E4-b 受众认知表型：
  历史认知改变：______
  框架借用：______
  批判意识：______
  → [一致/不一致]，[P等级]

E4-c 受众行为表型：
  [已采集/缺口/待方法成熟]
```

## 七、迭代终止条件

本提示词设计为可多轮迭代使用。分析者提供反馈后进入修正流程：

```
【收到反馈后的执行步骤】

Step 1：理解偏差来源——事实锚定错了？逻辑链条错了？信息缺失？
Step 2：补充检索——回到阶段一补充缺失信息，更新事实锚定底表
Step 3：重新推导——基于更新后的事实底表，从L1开始重新推导受影响层级
Step 4：输出修正版——标明修改部分、修改前版本、修改原因
Step 5：二次验证——重新执行闭环自校验
```

**迭代终止条件**：
1. 分析者确认"PASS"
2. 连续两轮反馈方向相反（标记为"不确定性"，保留争议）
3. 最多3轮迭代

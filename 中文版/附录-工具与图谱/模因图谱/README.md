## 文学作品模因图谱

一个通过流程保障深度的作品立场分析框架。输入作品名，自动完成五层分析（L1-L5）+ 四维交叉验证 + 反事实测试 + 终极判定。

核心创新：**把批判性思维编码为不可跳过的硬性规则**。滑梯退出、对称性强制、拮抗标注、置信度锚定——即使执行者没有批判性思维天赋，严格按流程走也能产出深度分析。

详见 [literature/README.md](literature/README.md)。

## 人类模因图谱

一个通过流程保障深度的**数字人引擎构建框架**。输入人名，自动完成 M1-M6 模因分析 + DIM-01~05 行为规则提取 + 场合激活映射 + 历史回测验证，最终生成可运行的数字人 SKILL.md。

核心创新：**将模因分析从静态分类升级为可运行的预测引擎**。对称性强制、反事实测试、滑梯退出——即使执行者没有心理学背景，严格按流程走也能构建出有预测力的数字人模型。

详见 [human/README.md](human/README.md) 和 [human/SKILL.md](human/SKILL.md)。

# AI分析工具：把理论变成武器

这本书不只是让你"知道"，还能让你"用"。这里介绍两个AI辅助工具，帮你把书里的方法用到实际生活中。

---

## 工具一：作品立场分析器

### 它能干什么

你输入一部作品的名字，它帮你分析：这部作品在暗中传递什么价值观？它在为谁说话？它是在叫醒你，还是在哄你睡？

### 怎么用

在Trae IDE里直接说：

> "分析《作品名》的立场"
> "作品立场 《作品名》"

### 它从六个角度分析

| 角度 | 看什么 |
|------|-------|
| 谁受益 | 这部作品服务于谁的利益？ |
| 情绪往哪走 | 读者的情绪能量被引向哪里？ |
| 操控手法 | 作品里有哪些操控读者的技巧？ |
| 表面vs里子 | 表面讲的故事和底层逻辑是不是两回事？ |
| 传播了什么"思想病毒" | 作品在传播哪些观念？这些观念为什么能活下来？ |
| 叫醒还是哄睡 | 作品是帮你看得更清，还是让你继续舒服地睡着？ |

### 什么都能分析

小说、电影、电视剧、公众号文章、短视频、广告、新闻报道——只要是传递信息的东西，都能分析。

## 工具二：模因图谱构建器

### 它能干什么

深度剖析一个人物或一部作品的"思想结构"，画出完整的"思想病毒"地图。

### 怎么用

在Trae IDE里直接说：

> "模因分析 《人物/作品名》"

### 它从五个角度分析

| 角度 | 看什么 |
|------|-------|
| 核心信念 | 最底层、不会动摇的信念是什么？ |
| 被植入的 | 哪些想法是外面塞进来的？ |
| 内部打架 | 哪些想法之间互相矛盾？ |
| 传播策略 | 这些想法怎么复制自己、感染别人？ |
| 软肋 | 哪些地方最容易被攻破？ |

---

## 使用建议

1. **先读书，再用工具**：工具的分析方法来自这本书，理解了理论再用，效果翻倍。
2. **从你熟悉的作品开始**：第一次分析选一部你已经很了解的作品，这样你能判断工具说得对不对。
3. **别只看结论**：分析过程中每一步的推理比最终结论更重要——那是你学习"开天眼"的过程。
4. **保持怀疑**：AI工具也可能有偏差。把工具的分析当作"一个视角"而不是"标准答案"。

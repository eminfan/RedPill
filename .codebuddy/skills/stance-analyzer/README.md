# 作品立场分析 · Stance Analyzer

> 输入作品名，自动完成完整立场分析（v5.8框架）。
> 分析一部作品在暗中传递什么价值观、为谁的利益说话、对读者是叫醒还是哄睡。

## 一句话

拆解作品在"暗中传递什么价值观、为谁的利益服务"——是让人更清醒（启蒙），还是让人更麻木（驯化）。

## 安装

```bash
npx skills add stance-analyzer
```

或在 Claude Code 中直接触发：输入「作品立场」「分析XX」「XX的立场」「stance XX」「XX在传递什么」等关键词即可。

## 用法

在 Claude Code 中，输入作品名即可。例如：

- 「分析《活着》」
- 「红楼梦的立场」
- 「stance 三体」
- 「鹿鼎记在传递什么」

分析框架会自动执行完整流程（5层分析+4维交叉验证+自鉴声明），无需用户了解框架细节。

## 分析输出示例

| 作品 | 类型 | 证据等级 | 标签 |
|------|------|---------|------|
| 《三国演义》 | A类·历史 | S级 | 弱驯化·顺从型+承受型 |
| 《活着》 | B类·现实主义 | A级 | 弱驯化·承受型 |
| 《红楼梦》 | F类·混合型 | A级 | 弱启蒙—弱驯化混合 |
| 《狂人日记》 | D类·寓言 | A级 | 强启蒙 |
| 《论持久战》 | E类·功能文本 | S级 | 强启蒙 |

## 框架概要

**四阶段执行**：
1. **检索校准** — 作品类型识别 + 事实锚定底表
2. **作品分析（五层）** — 素材层→结构层→功能层→调控层→演化层
3. **四维交叉验证** — 文本+传播+社会+效果证据
4. **最终判断** — 标签+自鉴声明

详见 [框架（当前版本）.md](框架（当前版本）.md)。

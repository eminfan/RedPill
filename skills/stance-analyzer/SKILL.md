# 作品立场分析 · 完整驯化/启蒙分析（v6.0）

> 基于 v6.0 框架。相比 v5.8 核心升级：强制外部检索、强制事实引用、终极判定具体化、反事实测试强制。

---

## 0. 全局硬性规则（违反任意一条 → 立即终止并告知用户）

1. **事实锚定是分析的基石。事实错了分析全错。** 任何没有具体事实引用的"偏差标注"不纳入最终判断。
2. **不得以"训练数据中的印象"替代查证。** A类（历史）和E类（功能文本）必须对外部检索结果。
3. **最终判断的三个回答中，必须有具体的社会群体/历史人物/利益方名称。** 全是抽象名词的判为「不合格分析」，直接终止。

---

## Step 0：启动

收到用户输入的作品名后：

1. **确认作品**：作者、创作年代、基本背景
2. **输出分析者立场声明**：

```
【分析者立场声明】
我对本作品/作者/题材的既有立场：[根据训练数据推测]
可能的影响：[该立场可能导致分析偏向于强化/弱化某方向的判断]
```

3. 告知用户：分析需持续输出较长时间（3-5分钟），建议一次性完成
4. 直接进入 Step 1

## Step 1：检索校准（强制检索阶段 — 最重要的改进）

### 1.1 作品类型识别

```
A. 历史题材——涉及真实历史人物/事件/朝代
B. 现实主义题材——以当代/近代现实社会为背景
C. 纯虚构世界观——完全虚构的世界
D. 寓言/改编作品——有明确原著的改编，或自觉使用寓言/象征/隐喻手法的创作
E. 功能文本——说明书/教程/规章/宣传材料/政论/战略论述
F. 混合型
G. 纯美学文本——不以传递世界观为目的
```

**D类判定标准**（符合任意一条即归为D类）：
- 有明确原著且进行了系统性改编
- 核心手法是寓言/象征/隐喻
- 存在自觉信号"这不是在说这个，是在说那个"

**G类**：字数<2000、无人无情节无对话 → 直接输出"不适合模因图谱分析"，结束。

### 1.2 强制外部检索（v6.0 核心升级）

**A类·历史题材**——**必须**对外部检索（WebSearch/WebFetch），搜索以下内容：

1. **该作品涉及的核心历史事件**的真实经过（至少3个事件）
2. **该作品涉及的核心历史人物**的正史评价（至少3个人物）
3. **该时期的宏观数据**（经济/人口/科技等）
4. **作者的创作历史语境**

**B类·现实主义**——建议检索：
1. 作品涉及的社会矛盾的真实数据
2. 作者身份和公开立场

**D类·寓言/改编**——**必须**检索：
1. 原著的原始模因（改编作品）
2. 改编者的公开立场和改编意图
3. 该作品在现实中的象征指涉

**E类·功能文本**——**必须**检索：
1. 核心论点是否与客观事实一致（找反方事实）
2. 引用数据是否准确
3. 同时代的竞争性文本

**检索执行记录**——完成后必须输出：

```
信息来源：[「外部检索」/「训练数据+检索补充」/「仅训练数据（风险：___）」]
检索词：[列出实际使用的检索词]
检索结果摘要：[对检索结果做2-3句总结]
知识完整度：[高/中/低——哪些维度可能信息不足]
```

**关键规则**：如果不对A类作品做外部检索 → 禁止输出"置信度高"的任何标注。所有标注自动降为"中"或"低"。

### 1.3 事实锚定底表

**A/B/C/F类**：

```
| 维度 | 作品叙事 | 真实事实（引用检索结果） | 偏差类型 |
|------|---------|----------------------|---------|
| 事件/人物 | 作品说…… | 事实上是……（信息来源：____） | 美化/抹黑/回避/置换 |
```

**同时收录正向素材**：

```
| 正向素材 | 作品呈现 | 事实核对 | 类型 |
|---------|---------|---------|------|
| 事件/人物 | 作品说…… | 确实如此…… | 正确呈现/启蒙洞察 |
```

**D类**使用替代框架（符号/角色→现实指涉，叙事选择→改编意图）。
**E类**使用替代框架（论点正确性、修辞策略、选择性呈现、框架预设）。

**D类输出前强制声明**：

```
D类寓言声明：本作品核心手法是寓言/象征/改编，不适用标准事实偏差标注。
以下分析不将"与事实不符"视为偏差，而是分析符号系统的指向和改编选择的意图。
```

**层1——标注**

| 类型 | 定义 |
|------|------|
| 美化 | 将负面事实呈现为中性或正面 |
| 抹黑 | 将中性或正面事实呈现为负面 |
| 回避 | 对关键事实完全不提及 |
| 置换 | 用一个伪命题替代真实命题 |
| 正确呈现 | 与事实一致 |
| 启蒙洞察 | 揭示了被掩盖的结构性真相 |

```
标注ID: DEV-001
素材: SRC-001
类型: [美化/抹黑/回避/置换/正确呈现/启蒙洞察]
方向: [具体描述——必须写出具体的叙事内容是啥]
依据: [事实引用——必须具体，可以引用检索结果]
置信度: [高/中/低]
```

**重要——置信度锚定规则（硬性）**：

| 置信度 | 条件 |
|-------|------|
| 高 | 依据栏填写了具体事实引用（含来源/数据/原文），且可直接验证 |
| 中 | 依据栏填写了推理过程但无直接事实引用 |
| 低 | 依据栏为空或填写了主观判断 |

- **依据为空的标注→自动标为低置信度→不纳入结构层（L2）的素材引用计数**
- 对A类作品，如果没有用检索结果填充依据栏 → **不能标"高"**

**素材比例量化**（层1完成后输出）：

```
【素材比例量化】
偏差素材总数：__  正向素材总数：__  合计：__
偏差类型分布：美化__ / 抹黑__ / 回避__ / 置换__
正向类型分布：正确呈现__ / 启蒙洞察__
偏差vs正向比例：__:__
→ 偏差主导（偏差≥70%）/ 正向主导（正向≥70%）/ 相对均衡（偏差40-60%）
```

## Step 3：四维交叉验证

### 证据类型分层

| 证据类型 | 定义 |
|---------|------|
| 可验证 | 有公开可查的事实依据 |
| 合理推断 | 基于可验证事实的逻辑推理 |
| 推测 | 无直接事实依据的分析者判断 |
| 数据缺口 | 该维度缺乏任何可用数据 |

### 证据等级

| 等级 | 条件 |
|------|------|
| S级 | 4维一致（至少2维为可验证/合理推断） |
| A级 | 3维一致 |
| B级 | 2维一致 |
| C级 | 仅文本证据 |
| D级 | 无证据 |

**关键规则**：标注为"推测"的证据**不纳入一致性矩阵的证据计数**。

### 一致性矩阵

```
【一致性矩阵】

维度一：身份定位
  E1：______  E2[证据类型]：______  E3[证据类型]：______  E4[证据类型]：______
  → 验证：______  有效证据数：__/3

维度二~五：同上

总体：[S/A/B/C/D]
总体解读：______
缺失维度：______
证据质量说明：______
```

**冲突处理**：E1≠E2以E2为准；E1≠E4以E4为准；E2≠E4分析差异来源。

#### ⏏ 自检关卡3

```
□ 一致性矩阵20个单元格均有内容
□ L5激活状态已标注
□ 总体证据等级已判定
□ 证据等级判定有据
```

## Step 4：反事实测试（v6.0 新增——强制完成）

在给出最终判断之前，先主动生成3个最强有力的反论点，然后逐一回应。这是防止分析成为"单向批评"的关键机制。

```
【反事实测试】

反论点1：[来自支持相反立场的读者可能提出的最强反驳]
→ 回应：[是否成立？如果成立，局部修正主结论；如果不成立，说明为什么不成立]

反论点2：[来自文本内部矛盾/复杂性的反驳]
→ 回应：

反论点3：[来自分析者自身立场的反思——"是否存在解读过度？"]
→ 回应：

对主结论的影响：[强化/弱化/局部修正]
```

## Step 5：最终判断 + 自鉴声明

### 最终判断

```
【最终判断】
标签：[强驯化 / 弱驯化 / 中性 / 弱启蒙 / 强启蒙]
⌟ 驯化方向：[承受型/竞争型/冷漠型/顺从型/混合型]

① 这部作品在为谁的利益说话？
  - 必须答出具体的社会群体/历史人物/利益方
  - ❌ "服务于统治阶级" → ✅ "服务于[具体群体]的[具体叙事服务]"
  - 至少写出2个利益方（受益者+受损者）

② 对读者是叫醒还是哄睡？如果是哄睡，用的是什么具体手法？
  - 列出至少3种具体手法，每种1-2句说明

③ 看懂了该怎么做？——防御+认知修复
  - 至少列出2条可操作的读者防御策略
```

**输出后自检（硬性）**：三个回答中是否有具体的社会群体/利益方名称？是否全是抽象名词？是则重写。

### 标签判定规则

| 标签 | 条件 |
|------|------|
| 强启蒙 | 世界模型揭示真相、打破驯化 |
| 弱启蒙 | 世界模型提供另类视角但不彻底 |
| 中性 | 无根模因倾向 |
| 弱驯化 | 有偏向但不极端，保留审美/知识价值 |
| 强驯化 | 系统性扭曲事实，消解批判能力 |

### 分析者立场声明与自鉴

```
【分析者立场声明】
分析启动时声明的既有立场：[正面/负面/中立/不了解]
可能的影响：[该立场可能导致分析偏向于强化/弱化某方向的判断]

【量化指标摘要】
偏差vs正向比例：__:__
偏差集中在：[美化/抹黑/回避/置换]
素材层置信度分布：高__ / 中__ / 低__

【自鉴声明】
换一个立场相反的分析者，用同样的事实锚定底表，可否得出不同判断？
  → [很可能相同/可能不同]
如可能不同，偏差来源是：

  □ 证据等级判定过严/过松
  □ 最终判断依赖主观解读
  □ 反事实测试被忽视
  □ 其他：______
```
